# Indlæring af kanaler til asymmetriske udsving

Dette afsnit er kun relevant, hvis alle tidligere trin er udført korrekt, og fjernlys eller tågelygter viser for lille eller ingen udsving.

Mulige årsager:

- Forkert nulstilling (trimmer, subtrim)
- Elektronisk begrænsning af kanalen
- Billig fjernbetjening

Nulstil om muligt alle indstillinger for den anvendte kanal, og start den normale indlæringsproces.

Hvis dette ikke er muligt, skal udstyringen for begge udsving indstilles separat.

Åbn LiveData-assistenten og fokuser på den nedenfor markerede værdi (ved brug af K2 værdien til venstre for denne). Denne værdi ændres ved tryk på knappen og bør være så tæt som muligt på følgende værdier:

- Nulstilling: 1500
- Niveau-knap: 1000 *(afhængigt af sender også 2000)*
- Positionslys: 1070 *(afhængigt af sender også 1930)*
- Minus-knap: 2000 *(afhængigt af sender også 1000)*
- Saddelkobling: 1930 *(afhængigt af sender også 1070)*

Jo større udstyringen er, desto længere bevæger værdierne sig væk fra 1500. Udsvinget for niveau-knappen bør være så langt væk som muligt, mens udsvinget for positionslys skal være tydeligt forskelligt. Det samme gælder for minus-knappen og saddelkoblingen.

Indstil nu udstyringen for begge udsving separat:

- Tryk på Setup+ og S4 for at indstille maksimal udstyring for det positive udsving.
- Kontroller værdierne for pad-knappen og positionslyset. Er de ens (f.eks. 1930 og 1925), reducer udstyringen med Setup+ og S3 og kontroller igen. Forskellen bør være mindst 50.
- Gentag processen for det negative udsving med Setup- og S-knapperne. Kontroller værdierne for minus-knappen og saddelkoblingen.
