# Funktionsomfang

Tasterne kan udløse forskellige funktioner afhængigt af trykvarigheden:

- **Kort tryk:** Orange symboler
- **Langt tryk:** Hvide symboler
- **Ekstralag (skiftbart med venstre øverste tast):** Grå symboler, kort/langt tryk

## Øverste tastelinje

| Kort tryk | Langt tryk |
| --- | --- |
| Padebene | Motor Start/Stop |
| Blinklys venstre | Servo 1 |
| Nødblink | Partymodus |
| Blinklys højre | Servo 2 |
| Opgear | Fjernlysblink |
| Nedgear | Horn |

## Nederste tastelinje

| Kort tryk | Langt tryk |
| --- | --- |
| Positionslys | Støtter op |
| Nærlys | Støtter ned |
| Fjernlys | Kasse op |
| Tåge-/baglys | Kasse ned |
| Rundtlys | Rampe op |
| Trækstang | Rampe ned |

**Padebene:** Ved at trykke på den venstre øverste tast skiftes til ekstralagets funktionsniveau. Her kan op til 22 ekstra funktioner (hver elleve kort/langt tryk) styres. Funktionerne er arrangeret mod uret startende nederst til venstre (grå symboler).
