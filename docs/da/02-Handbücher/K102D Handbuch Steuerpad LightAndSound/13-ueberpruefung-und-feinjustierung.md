# Kontrol og finjustering

Når styring og kanaler er korrekt indlært, bør padden fungere. Test hver knap og kontroller i LiveData, om de tilsvarende flueben bliver sat.

**Tip:** Brug LiveData-visningen og ignorer først lysene/funktionerne på modellen for at udelukke forvirring på grund af programmeringsfejl.

Ved billige fjernbetjeninger kan det være nødvendigt at finjustere evalueringen af padsignalerne.
