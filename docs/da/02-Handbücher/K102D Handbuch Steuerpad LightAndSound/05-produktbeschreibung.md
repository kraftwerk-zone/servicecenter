# Produktbeskrivelse

Styringspadden muliggør styring af alle lysfunktioner samt roterende lys og sædekobling via kort trykfunktion (orange symboler). Derudover kan funktioner som påhængsstøtter, tipkasse, rampe, gearkasse, to servoer, fjernlysblink, horn, alt tændt/slukket samt motor start/stop styres ved langt tryk (hvide symboler).

Via et lag-skift (knap øverst til venstre) kan op til 22 ekstra funktioner styres (hver elleve via kort og langt tryk).

Styringspadden er kompatibel med alle fjernbetjeningssendere og optager kun én kanal. Ideelt er brugen af en ledig udvidelseskanal. Hvis der ikke er ledige udvidelseskanaler, kan en allerede optaget styrekanal bruges, men den vil så ikke længere være tilgængelig til andre anvendelser.

Signalbehandlingen sker i de centrale styreenheder KLM 4/0, KLM 4/12 – 500 eller KLM 4/16. Også ældre KLM lysassistenter er efter en opdatering kompatible med det aktuelle ControlPanel og kan bruges uden begrænsninger.

Ibrugtagningen sker i fem trin, som forklares detaljeret nedenfor:

1. Installation i fjernbetjeningssenderen  
2. Konfiguration af KLM  
3. Indstilling af udstyring  
4. Indlæring af kanaler  
5. Valgfrit: Finjustering  

Bemærk: Trin 3 og 4 skal eventuelt gentages flere gange, indtil den optimale udstyring er fundet.
