# Indlæring af kanaler til asymmetriske udsving

Dette afsnit er kun relevant, hvis alle tidligere trin er udført korrekt, og fjernlys eller tågelygter viser for små eller ingen udsving.

Mulige årsager:

- Forkert midterstilling (trimmer, subtrim)
- Elektronisk begrænsning af kanalen
- Billig fjernbetjening med ikke-lineær styrekurve

Nulstil om muligt alle indstillinger for den anvendte kanal, og start den normale indlæringsproces igen.

Hvis dette ikke er muligt, skal udstyringen for positive og negative udsving indstilles separat.

Åbn LiveData-assistenten, og fokuser på den nedenfor markerede værdi (ved brug af K2 på værdien til venstre). Denne værdi ændrer sig ved tastetryk og bør være så tæt som muligt på følgende målværdier:

| Funktion | Værdi (µs) |
| --- | --- |
| Midterstilling | 1500 |
| Fladtast | 1000 \* |
| Positionslys | 1070 \*\* |
| Minus-tast | 2000 \* |
| Saddelkobling | 1930 \*\* |

\* Afhængigt af fjernbetjeningen kan fladtast og minus-tast være byttet om.

\*\* Det samme gælder for positionslys og saddelkobling.

Jo større udstyring, desto længere bevæger værdierne sig væk fra 1500 µs ved tastetryk. Udsvinget for fladtasten bør være så langt væk som muligt, mens udsvinget for positionslys dog skal være tydeligt forskelligt. Det samme gælder for minus-tasten og saddelkoblingen.

Indstil udstyringen for begge udsving separat:

- Tryk på **Setup+** og **S4** for at indstille maksimal udstyring for det positive udsving.
- Kontroller værdierne for fladtast og positionslys. Er værdierne for ens (f.eks. 1930 og 1925), reducer udstyringen med **Setup+** og **S3** og kontroller igen. Forskellen bør være mindst 50 µs.
- Gentag processen for det negative udsving med **Setup-** og de tilsvarende S-taster. Kontroller værdierne for minus-tast og saddelkobling.
