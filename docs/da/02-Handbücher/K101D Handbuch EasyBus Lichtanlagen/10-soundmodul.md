# Soundmodul

Soundmoduler kan drives direkte på modtageren. For styring via EasyBus skal de tilsluttes to servo-udgange (direkte på KLM 4/16 eller på KSB 2/4).

## Styring af soundmodulet

Med styrepanelet Lys & Sound er der ikke behov for yderligere indstillinger. Hornknappen styrer hornet, og lagknappen starter eller stopper motoren.

De ekstra kanaler K1 og K2 (eller op til K10 ved brug af IBUS, SBUS, SUMD osv.) kan tilpasses i systemindstillingerne.

## Vigtige indstillinger

- **Motoren starter straks:** Afhængigt af om soundmodulet starter motoren straks, skal systemindstillingen „Motoren starter straks [aktiv]“ aktiveres eller deaktiveres.
- **Advarsel:** Kørselmotoren er i alle tilfælde aktiv!
- **Lys blinker:** Hver belysningsplade og kontaktudvidelse (KLB) har systemindstillinger for „Start-effekt“ (blink ved start) og „Motor stop lysreduktion“ (dæmpning ved motorstop). Brug sidstnævnte kun, hvis der ikke styres komponenter med integreret elektronik.
- Blinkets varighed kan tilpasses til soundmodulets startvarighed med „Motor start varighed“.
