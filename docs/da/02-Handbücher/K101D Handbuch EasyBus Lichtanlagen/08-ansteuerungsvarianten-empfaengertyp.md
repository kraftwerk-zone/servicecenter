# Styrevarianter / Modtager type

Styretypen kan vælges i systemindstillingerne for KLM 4/16 under „Modtagertype“.

## Normal 4xPWM

KLM 4/16 tilsluttes op til fire kanaler på modtageren. Kanalerne skal indlæres manuelt eller med ControlPanel for at definere midterstilling, endepositioner og rotationsretninger.

## CPPM

Hvis din modtager har en CPPM-udgang (summesignalet), kan denne forbindes til CPPM-indgangen på KLM. Aktiver eventuelt CPPM-udgangen på modtageren (se fjernstyringsanlæggets manual).

Kanalfordelingen sker via LiveData-visningen ved at bevæge styrestikkene én efter én og tildele de tilsvarende kanaler.

Efter tildelingen skal kanalerne indlæres som sædvanligt.

## I-BUS

For modtagere med I-BUS-udgang (f.eks. Turnigy) gælder samme fremgangsmåde som ved CPPM.

## S-BUS

For modtagere med S-BUS-udgang (f.eks. FUTABA) gælder samme fremgangsmåde som ved CPPM.
