# Indstillinger

Vores EasyBus-systemer tilbyder to typer indstillinger:

- **Systemindstillinger:** Gælder for hele modulet eller systemet (f.eks. blinkinterval, modtager-type, bremselystype).
- **Udgangsindstillinger:** Gælder kun for den enkelte udgang (se kapitel 7 „Funktioner“).

## Åbn systemindstillinger

Via tandhjulsikonet eller menuen „System“ → „Systemindstillinger“.
