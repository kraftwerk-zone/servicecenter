## Servofunktioner

| Funktion | Beskrivelse |
| --- | --- |
| Proportional | Styring af servo eller motorregulator med en proportional kanal. |
| Proportional med kontakt | Styring med proportional kanal, aktiv kun ved bestemt kontaktfunktion (f.eks. styring slukket ved løfteaksel op). |
| To positioner | Servo kører fra position 1 til position 2 ved aktivering af kontaktfunktionen. |
| Tre positioner | Servo kører fra position 2 til position 1 eller 3 afhængigt af aktiveret kontaktfunktion. |
| Sadelplade | Styring af sadelplade (gearskifte). |

## Motorfunktioner

| Funktion | Beskrivelse |
| --- | --- |
| Proportional | Styring af motor med proportional kanal. |
| Proportional med kontakt | Motorstyring kun ved aktiveret kontaktfunktion. |
| Sadelstøtter | Motorstyring ved valg af sadelstøttefunktion. |
| Kasse | Motorstyring ved valg af kassefunktion. |
| Tip | Motorstyring ved valg af tipfunktion. |
