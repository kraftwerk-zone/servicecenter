# Kørselsadfærd

## Type af kørselsregulator

Grundlæggende er enhver kørselsregulator kompatibel med EasyBus. For korrekt styring af bremse- og baklys skal den rigtige type kørselsregulator indstilles:

- **Enkel:** Bakning starter straks, når styrepinden trækkes bagud.
- **Bremse separat:** Først bremse, derefter bakning ved at trække igen.
- **Standard fartpilot:** Forsinkelse går direkte over i bakning.
- **Servonaut fartpilot:** Gearskift via K1, servo-udgang indstilles til „Proportionalfunktion K1“.
- **Servonaut tomgang:** Som Servonaut fartpilot, men med tomgangsposition mellem fremad og bak.
- **Servonaut K40:** Bremse- og baklys styres i K40-stil.
- **Analog bremse-/baklys:** Skifteudgange fra en kørselsregulator tilføres analogt (se kapitel 11.4).

## Intelligent bremselys

Bremselyset tænder allerede ved deceleration, ikke først ved bakning. Følsomhed og efterlysningstid kan indstilles i systemindstillingerne.

## Styretype

Som standard skal blinklyset tændes manuelt og nulstilles automatisk. Alternativt kan „Automatisk blinklys“ vælges, som aktiveres automatisk ved sving. „Blinklystærskel“ definerer aktiveringsgrænsen.

## Kurvelystype

Kurvelysadfærden kan indstilles via parameteren „Kurvelystype“. Valgmulighederne taler for sig selv.
