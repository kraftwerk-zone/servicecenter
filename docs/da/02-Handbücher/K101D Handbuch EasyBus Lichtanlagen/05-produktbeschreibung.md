# Produktbeskrivelse

KLM 4/16 tilsluttes op til fire kanaler på modtageren:

- Gas kanal til styring af bak- og bremselys
- Styrekanal til kurvelys og automatisk blinklysgendannelse
- Op til to ekstra kanaler til lys- og ekstra funktioner

Alternativt kan KLM 4/16 styres via CPPM, I-BUS eller S-BUS (hvis understøttet af modtageren) eller en datagrænseflade.

Af de 16 afbryderudgange er otte højstrømsudgange (op til 500 mA), de øvrige otte har 180 ohm modstande og kan direkte styre LED'er.

Hver udgang kan frit indstilles i funktion (f.eks. positionslys, bremselys, blinklys) og adfærd (tænd/sluk hastighed, lysstyrke). Udgang 8 kan konfigureres som infrarød sender (sendediode medfølger).

De fire servo-udgange kan også indstilles i drejeretning, endepositioner og hastighed.

KLM 4/16 er kompatibel med alle fartregulatorer, lydmoduler og fjernstyringsanlæg. Den understøtter Multiswitch-moduler fra Graupner og Robbe samt vores styrepads grund- og ekstra funktioner og det nye Light And Sound Pad.
