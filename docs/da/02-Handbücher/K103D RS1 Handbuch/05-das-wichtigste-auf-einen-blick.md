# Det vigtigste på et øjeblik

## Softwarestatus

Download altid den nyeste version af ControlPanel, og forbind modulet til din PC. Alle tilgængelige opdateringer, inklusive nye funktioner og fejlrettelser, installeres automatisk.

## Status-LED

- **Grøn (kort blink hvert 2. sekund):** Normal drift
- **Gul (langsommere blink):** StandBy-tilstand, ingen styresignaler modtages
- **Hurtigt rødt blink:** Underspænding – advarselsblink aktiv, motoren slukkes
- **Langsomt rødt eller andet farvet blink:** Kontakt venligst vores support

## StandBy-knap

Med knappen kan du tænde og slukke modellen eller sætte den i StandBy-tilstand:

- **StandBy-tilstand:** Alle funktioner kører videre, men der modtages ingen styresignaler fra fjernbetjeningen – ideelt til demonstrationer.
- **OFF-position:** Regulator, servoer og busmoduler er slukket. Bemærk: Der kan stadig være en lille strømstrøm, som kan føre til dyb afladning af batteriet. Afbryd derfor altid batteriet, senest ved dagens slutning.
