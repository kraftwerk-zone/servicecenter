# Lydtyper

![](images/image\_045.png)

image59.png

- **Motorsound Standard:** Acceleration og kontinuerlig kørsel
- **Motorsound Last:** Acceleration og kørsel i lasttilstand
- **Motorsound Verzögerung:** Køretøjet sænker farten
- **Motorsound Motorbremse:** Køretøjet sænker farten med motorbremse
- **Motorsound Lastwechsel:** Engangslyd ved skift af kørselsstatus
- **Motor Start/Stop:** Start- og stoplyd, lastafhængigt skiftbar
- **Motor Fehlstart:** Fejlstart-lyd
- **Turbo:** Turbofløjten
- **Schaltgeräusch:** Lyd ved gearskift
- **Luftbremse An/Aus:** Lyd fra trykluftbremsen
- **Rückfahrwarner:** Ved bakning
- **Hupe/Horn/Fanfare:** Kan skiftes via horn
- **Zusatzsound:** Brugerdefinerede lyde
- **Einschaltsound:** Efter fuld opstart af modulet
- **Anfahrsound:** Ved igangsætning (ofte ikke nødvendig)
- **Anhaltesound:** Ved standsning
- **MP3:** Kun én MP3 kan afspilles ad gangen
