# Tekniske data

## Fartregulator

|  |  |
| --- | --- |
| Driftsspænding | 6 - 20 V |
| BEC | 6 V / 3 A |
| Skiftefrekvens | justerbar op til 32 kHz |
| Koblingsfunktion | ja |
| Gearkasseunderstøttelse | ja (opdatering planlagt) |
| Inerti-simulering | valgfri |
| Fartpilot | valgfri (opdatering planlagt) |
| Beskyttelsesfunktioner | Batteriunderspænding, motorkortslutning, overtemperatur |
| Servofunktioner | Proportionalfunktion, skiftet, to positioner, tre positioner (f.eks. saddelplade, påhængsstøtter, motor lad/kipper, motor rampe/platform, gearkasse) |

## Lyd

|  |  |
| --- | --- |
| Antal lyde | ubegrænset |
| Samtidigt afspillelige lyde | ca. 10 |
| Afspilningstilstande | tastetryk, loop, én gang, permanent, tilfældig, stop-punkter |
| Understøttede formater | WAV, MP3 |
| Understøttet sample rate | Alle (automatisk konvertering til 44.100 Hz) |
| Lydstyrkekontrol | potentiometer, via kanal, via software |
| SD-kort | maks. 32 GB, større kort kan formateres til 32 GB FAT32 |
