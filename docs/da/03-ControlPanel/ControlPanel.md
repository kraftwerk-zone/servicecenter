## Beta2025.1 – ControlPanel (08.07.2025)
**Download:** https://kraftwerk-zone.com/_downloads/ControlPanel_jre_202510.exe  
**Størrelse:** 151 MB

### Ændringer
- PremiumSounds tilgængelige igen, nu også med ordrenummer i stedet for adgangskode
- Infrarød kan tændes/slukkes, træghed kan tændes/slukkes
- Kanaler og funktioner kan navngives via systemindstillingerne
- Sequencer forberedt
- Virtuelt pad forberedt
- Ny firmware til servoindgange (C007 problem)
- RS1 blinker lejlighedsvis violet og nægter service → rettet
- Udgange blev ikke vist → rettet
- Ryk i servos for SBUS → rettet
- Bemærkning til SBUS-brugere: Hvis der ikke vises "RS1 PWM indgange (interne)" → skift gas/data til normal modtagerudgang, genstart, genforbind
