# Aprendizaje de los canales

El aprendizaje es fundamental para el correcto funcionamiento de los sistemas KraftwerK EasyBus. Después de cada cambio en el sistema de control remoto (desplazamientos, direcciones de giro, posiciones centrales), los canales deben ser reprogramados.

Utilice exclusivamente el ControlPanel para ajustar servos y motores, no el menú de su sistema de control remoto.

1. Encender el control remoto.
2. Elevar el modelo para que ninguna rueda motriz toque el suelo.
3. Llevar el canal de gas y de dirección a su extremo máximo.
4. Encender el modelo – el LED rojo se enciende y confirma el modo de aprendizaje.
5. Volver a colocar las palancas en la posición central.

Señales LED durante el aprendizaje:

- 1x rojo corto: Palanca de gas – extremo máximo de gas, luego extremo máximo de freno
- 2x rojo corto: Palanca de dirección – extremo máximo a la izquierda, luego a la derecha
- 3x rojo corto: Canal adicional K1 – extremo máximo a la izquierda, luego a la derecha
- 4x rojo corto: Canal adicional K2 – extremo máximo arriba, luego abajo
- 5x rojo corto: Canal adicional K3 – extremo máximo arriba, luego abajo

Después de que todas las palancas vuelvan a la posición central, el LED se enciende en rojo fijo – aprendizaje completado. El sistema se reinicia y señala con un parpadeo verde doble que está listo para operar.
