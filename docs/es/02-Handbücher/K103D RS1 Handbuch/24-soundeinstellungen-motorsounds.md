## Visualización del sonido

![](images/image\_049.png)

image63.png

- **RPM mín:** Revoluciones a partir de las cuales el sonido se va introduciendo lentamente
- **RPM ref:** Revoluciones base a las que el sonido se reproduce a volumen completo
- **RPM máx:** Revoluciones hasta las cuales el sonido se va atenuando lentamente

Los sonidos se introducen y atenúan adaptándose a las revoluciones, lo que genera una imagen sonora realista.
