## Botones superiores

- **Disparador:** 1. Flanco ascendente, 2. Flanco descendente, 3. Ambos flancos
- **Tipo de reproducción:** 1. Bucle, 2. Reproducir siempre, 3. Volver al inicio o continuar reproduciendo

Todas las combinaciones son posibles.

## Paradas

Para cada sonido se pueden definir puntos de parada en segundos. Con la función "reproducir siempre" activada, el sonido se reproduce hasta el siguiente punto de parada o continúa si la función sigue activa.

## Aleatorio

El sonido se inicia cada x a y segundos, siempre que esté activo.
