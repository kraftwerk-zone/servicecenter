# Datos técnicos

## Regulador de velocidad

|  |  |
| --- | --- |
| Tensión de funcionamiento | 6 - 20 V |
| BEC | 6 V / 3 A |
| Frecuencia de conmutación | ajustable hasta 32 kHz |
| Función de embrague | sí |
| Soporte para caja de cambios | sí (actualización prevista) |
| Simulación de inercia | opcional |
| Control de crucero | opcional (actualización prevista) |
| Funciones de protección | Subtensión de batería, cortocircuito del motor, sobretemperatura |
| Funciones de servo | Función proporcional, conmutada, dos posiciones, tres posiciones (p. ej. placa de silla, soportes de remolque, motor de tolva/volquete, motor de rampa/plataforma, caja de cambios) |

## Sonido

|  |  |
| --- | --- |
| Número de sonidos | ilimitado |
| Sonidos reproducibles simultáneamente | aprox. 10 |
| Modos de reproducción | pulsación de tecla, bucle, una vez, permanente, aleatorio, puntos de parada |
| Formatos utilizables | WAV, MP3 |
| Frecuencia de muestreo utilizables | Todas (conversión automática a 44.100 Hz) |
| Control de volumen | Potenciómetro, por canal, por software |
| Tarjeta SD | máx. 32 GB, tarjetas más grandes pueden formatearse a 32 GB FAT32 |
