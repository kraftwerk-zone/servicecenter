# Conexiones

- 5 entradas de servo – a Gas/Data también se puede conectar una señal IBUS/SBUS/SUMD (detección automática)
- 5 salidas de servo (ajustables, configuración de fábrica ver arriba)
- Interruptor de Apagado/Encendido/StandBy
- Jumper de StandBy para configurar el modo de sonido y control
- 6 salidas EasyBus
- Conector de expansión I2C
- Conector Micro-USB
- Ranura para tarjeta SD (máx. 32 GB, formateada en FAT32)
- Conector para altavoz (4 ohmios o más)
- Salidas de conmutación de 10A
- LED de estado
- 4 pines de expansión para módulos enchufables
