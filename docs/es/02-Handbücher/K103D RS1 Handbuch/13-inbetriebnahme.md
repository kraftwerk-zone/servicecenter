# Puesta en marcha

El RS1 debe conectarse a cinco canales del receptor. Mediante un jumper se puede configurar el uso de un panel de control para luz y sonido.

## Panel de control en K3 (jumper derecho colocado)

- El freno del motor está activo mientras K2 se mantenga presionado hacia arriba (si el sonido del modelo lo soporta).
- Si se tira de K2 hacia abajo y luego se pulsa hacia la derecha, la bocina cambia a claxon y fanfarria (si está soportado).

## Control sin panel de control (jumper superior no colocado)

La asignación de canales es más compacta aquí. Acelerador, dirección, volumen y carga se controlan como al usar el panel de control.

- Freno del motor como función de memoria: un toque corto activa/desactiva el freno del motor.
- Mantener pulsado hacia arriba arranca/detiene el motor.
- Un toque corto activa la bocina.
- Si se tira de K2 hacia abajo y luego se pulsa hacia la derecha, la bocina cambia a claxon y fanfarria.

## Nota sobre el uso de un pulsador en K3

Si se usa un pulsador para K3, solo se pueden activar funciones completas. Con mezcladores adecuados en el radiocontrol también se pueden realizar funciones parciales.
