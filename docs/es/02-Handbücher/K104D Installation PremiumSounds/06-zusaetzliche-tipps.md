# Consejos adicionales

- Guarde sus datos de usuario de forma segura para tener acceso en todo momento a sus sonidos comprados.
- Aproveche las escalas de precios si desea equipar varios módulos con los mismos sonidos para ahorrar costos.
- Al cambiar la tarjeta SD o el módulo, los sonidos se pueden reinstalar sin problemas, siempre que la licencia sea válida.
