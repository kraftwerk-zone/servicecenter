# Indicaciones para la resolución de problemas

- Asegúrese de que la versión del firmware del módulo de sonido esté actualizada (al menos 2024.3).
- Verifique la conexión USB entre el PC y el módulo de sonido si el módulo no es reconocido.
- Si no se puede instalar el sonido, compruebe si la licencia ya está asignada a otro módulo.
- En caso de problemas para iniciar sesión en el administrador de sonido premium, revise sus datos de acceso y la conexión a Internet.
