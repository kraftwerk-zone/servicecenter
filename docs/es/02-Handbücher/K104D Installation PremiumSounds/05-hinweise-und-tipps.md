# Consejos y recomendaciones

- **Actualización de firmware:** Asegúrese de que su módulo de sonido siempre funcione con la versión más reciente del firmware para evitar problemas de compatibilidad.
- **Instalación múltiple:** Si desea utilizar un sonido en varios módulos, consulte las escalas de precios disponibles antes de realizar el pedido.
- **Cambio de tarjeta SD:** Al cambiar la tarjeta SD, el sonido debe instalarse nuevamente, tal como se describe en el paso 4.
- **Soporte:** En caso de problemas o preguntas, nuestro servicio de atención al cliente está disponible por teléfono o correo electrónico.
