# Variantes de control / Tipo de receptor

El tipo de control se puede seleccionar en la configuración del sistema del KLM 4/16 bajo "Tipo de receptor".

## Normal 4xPWM

El KLM 4/16 se conecta a hasta cuatro canales del receptor. Los canales deben ser configurados manualmente o con el ControlPanel para definir la posición central, los recorridos finales y las direcciones de giro.

## CPPM

Si su receptor ofrece una salida CPPM (señal sumatoria), esta puede conectarse a la entrada CPPM del KLM. Active la salida CPPM en el receptor si es necesario (consulte el manual del sistema de radio).

La asignación de canales se realiza a través de la vista de datos en vivo, moviendo las palancas una tras otra y asignando los canales correspondientes.

Después de la asignación, los canales deben ser configurados como de costumbre.

## I-BUS

Para receptores con salida I-BUS (por ejemplo, Turnigy) se aplica el mismo procedimiento que para CPPM.

## S-BUS

Para receptores con salida S-BUS (por ejemplo, FUTABA) se aplica el mismo procedimiento que para CPPM.
