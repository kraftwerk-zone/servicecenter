# Información del dispositivo

Muestra información básica sobre los módulos. Con "Mostrar dispositivo" se activan las primeras tres salidas, útil para varias expansiones universales sin asignación de nombre.

Con "Reiniciar" se reinicia el módulo (similar a desconectar y volver a conectar). "Configuración de fábrica" restablece a la configuración de fábrica, "Actualizar" fuerza una actualización.

Si hay una nueva versión disponible, el ControlPanel lo indica al iniciar y ofrece la actualización. La actualización manual es posible a través de la información del dispositivo.
