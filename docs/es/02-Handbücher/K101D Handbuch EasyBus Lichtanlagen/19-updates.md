# Actualizaciones

Después de conectar, el ControlPanel ofrece actualizaciones disponibles para los módulos de bus.

![](images/image\_065.png)

image6.png

**Consejos para la actualización:**

- Actualizar siempre un módulo a la vez para evitar errores.
- Evitar interrupciones de corriente durante la actualización.
- Desconectar servos/motores en caso de sacudidas o zumbidos.
- Reiniciar regularmente el modelo y el ControlPanel, especialmente ante retrasos inusuales.

Las actualizaciones pueden restablecer los módulos a la configuración de fábrica. El ControlPanel guarda automáticamente todas las configuraciones en el disco duro para que puedan recuperarse posteriormente.
