# Ajustes importantes

## Motor arranca inmediatamente

Dependiendo del módulo de sonido, debe ajustarse la configuración del sistema «Motor arranca inmediatamente [activo]»:

- Módulo de sonido arranca el motor inmediatamente: marcar la casilla.
- El motor debe arrancarse manualmente: desmarcar la casilla.

**Precaución:** ¡El motor de tracción está activo en cualquier caso!

## Las luces parpadean

Cada placa de iluminación y extensión de interruptor (KLB) tiene configuraciones del sistema para «Efecto de arranque» (parpadeo al arrancar) y «Reducción de luz al parar motor» (reducción al detener el motor). Use esta última opción solo si no hay componentes con electrónica integrada conectados.

La duración del parpadeo puede ajustarse con «Duración de arranque del motor» para que coincida con la duración del sonido de arranque.
