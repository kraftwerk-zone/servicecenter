# Opciones de configuración

Nuestros sistemas EasyBus ofrecen dos tipos de configuraciones:

- **Configuración del sistema:** Aplica para todo el módulo o sistema (p. ej., intervalo de parpadeo, tipo de receptor, tipo de luz de freno).
- **Configuración de salida:** Aplica solo para la salida respectiva (ver capítulo 7 "Funciones").

## Abrir configuración del sistema

A través del icono de engranaje o el menú "Sistema" → "Configuración del sistema".

| Nombre | Descripción |
| --- | --- |
| Retardo de inicio [s] | Retrasa el inicio del sistema para considerar los tiempos de arranque de los controles remotos. |
| Modo de aprendizaje [activo] | Activa/desactiva el modo de aprendizaje manual mediante el acelerador y el joystick de dirección. El aprendizaje con ControlPanel permanece activo. |
| Infrarrojo [encendido] | Cambia la salida 8 a modo infrarrojo. |
| Intermitente Encendido/Apagado | Define el intervalo de parpadeo en pasos de 10 ms (p. ej., 500 ms = 0,5 s). |
| Tipo de receptor | Ver capítulo 8 "Variantes de control / Tipo de receptor". |
| Tipo de controlador de velocidad | Ver capítulo 9.1 "Tipo de controlador de velocidad". |
| Intermitente de emergencia al marcha atrás | Activación automática del intermitente de emergencia al ir en reversa. |
| Sensibilidad de luz de freno | Ver capítulo 9.2 "Luz de freno inteligente". |
| Duración de luz de freno | Ver capítulo 9.2 "Luz de freno inteligente". |
| Tipo de dirección | Ver capítulo 9.3 "Tipo de dirección". |
| Umbral del intermitente | Ver capítulo 9.3 "Tipo de dirección". |
| Tipo de luz de curva | Ver capítulo 0 "Tipo de luz de curva". |
| Motor arranca inmediatamente [activo] | Ver capítulo 10 "Módulo de sonido". |
| Duración de arranque del motor | Ver capítulo 10 "Módulo de sonido". |
| Botón Todo Encendido/Apagado del pad | Configura el comportamiento del botón "Todo Encendido/Apagado" en los pads de control. |
| Pad Luz y Sonido Luz alta | Ver manual del pad de control Luz y Sonido. |
| Pad Luz y Sonido Faros antiniebla | Ver manual del pad de control Luz y Sonido. |
| Asignación K1 – K12 | Ver capítulo 13.6 "Personalizado". |
