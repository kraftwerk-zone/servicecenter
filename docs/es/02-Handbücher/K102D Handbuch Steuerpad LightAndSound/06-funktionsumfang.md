# Alcance funcional

Las teclas pueden activar diferentes funciones según la duración de la pulsación:

- **Pulsación corta:** Símbolos naranjas
- **Pulsación larga:** Símbolos blancos
- **Nivel adicional (conmutado con la tecla superior izquierda):** Símbolos grises, cada uno corto/largo

## Fila superior de teclas

| Pulsación corta | Pulsación larga |
| --- | --- |
| Nivel del panel | Motor encendido/apagado |
| Intermitente izquierdo | Servo 1 |
| Intermitente de emergencia | Modo fiesta |
| Intermitente derecho | Servo 2 |
| Subir marcha | Luz de destello |
| Bajar marcha | Bocina |

## Fila inferior de teclas

| Pulsación corta | Pulsación larga |
| --- | --- |
| Luz de posición | Soportes arriba |
| Luz de cruce | Soportes abajo |
| Luz de carretera | Caja basculante arriba |
| Faro antiniebla/luz trasera antiniebla | Caja basculante abajo |
| Luz de emergencia | Rampa arriba |
| Acoplamiento del semirremolque | Rampa abajo |

**Nivel del panel:** Al pulsar la tecla superior izquierda se cambia al nivel de funciones adicionales. Allí se pueden controlar hasta 22 funciones adicionales (once cortas/largas cada una). Las funciones están dispuestas en sentido contrario a las agujas del reloj comenzando desde la parte inferior izquierda (símbolos grises).
