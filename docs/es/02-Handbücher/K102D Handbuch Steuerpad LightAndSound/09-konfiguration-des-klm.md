# Configuración del KLM

![](images/image\_013.png)

image15.png

El panel de control puede conectarse a K1 o K2. Conecte el modelo por USB, encienda el control remoto y el modelo, y abra el ControlPanel.

Abra la configuración del sistema a través del icono de engranaje o el menú "Sistema". Asegúrese de que en la lista desplegable superior esté seleccionado el KLM.

Cambie la "Asignación K1" o "Asignación K2" a "Pad Light And Sound".

**Consejo:** Si el control de capas está activado para K1, la asignación para K2 aparece deshabilitada, ya que este canal se usa para el cambio de capas. Si desea usar el panel en K2, seleccione otra opción para K1, por ejemplo, "Personalizado".

Después de guardar, el KLM estará configurado para el panel de control.

**Consejo:** Si no está seguro en qué canal está conectado el panel, la vista de datos en vivo del ControlPanel puede ayudar.
