# Configuración del KLM

![](images/image\_013.png)

image15.png

El panel de control puede usarse en los canales K1 o K2. Conecte el modelo por USB, encienda el control remoto y el modelo, y abra el ControlPanel.

Abra la configuración del sistema a través del icono de engranaje o el menú "Sistema". Asegúrese de que el KLM esté seleccionado en la lista desplegable superior.

Cambie la configuración "Asignación K1" o "Asignación K2" a "Pad Light And Sound".

**Consejo:** Si el control de niveles está activado para K1, la asignación para K2 aparece atenuada, ya que este canal se necesita para el cambio de niveles. Si desea usar el panel en K2, seleccione otra asignación para K1, por ejemplo, "Personalizado".

Después de guardar, el KLM estará configurado para el panel de control.

**Consejo:** Si no está seguro en qué canal está conectado el panel, la vista de datos en vivo en el ControlPanel puede ayudar.
