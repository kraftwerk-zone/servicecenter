# Funciones centrales temporizadas

Las funciones centrales temporizadas como el intermitente se controlan mediante dos funciones digitales:

- Información sobre si la función está activada o desactivada en general
- Impulso que enciende y apaga la lámpara

Para controlar el intermitente, basta con establecer la información del impulso. La otra información puede usarse para mezcladores y otros interruptores, por ejemplo, para cambiar a una segunda configuración cuando el intermitente está activado.
