# Preguntas frecuentes - RS1

---

## RS1 se queda atascado en un bucle al iniciar.

Si el RS1 se reinicia continuamente en un bucle, generalmente se debe a un firmware muy antiguo combinado con una tarjeta SD defectuosa.

**Pasos para solucionar**

1. Cambiar la tarjeta SD (máx. 32 GB, se recomienda FAT32)  
2. Instalar la versión más reciente del ControlPanel  
3. Copiar los contenidos de  
   `Dokumente\KraftwerK\ControlPanel\sdcard_original`  
   a la nueva tarjeta SD  
4. Reiniciar el RS1  
5. Reiniciar el ControlPanel  
6. Realizar todas las actualizaciones disponibles  

---

## RS1 interrumpe las actualizaciones – ¿por qué?

Si una actualización no se completa, a menudo falta el directorio de firmware en la tarjeta SD.

**Cómo solucionar el problema**

1. Abrir la tarjeta SD directamente en el PC  
   o en el ControlPanel bajo  
   **Configuración de sonido → Abrir tarjeta SD**  
2. Comprobar si existe una carpeta **Firmware** en la tarjeta SD  
3. Si no existe: crear una nueva carpeta **Firmware**  
4. Reiniciar el RS1  
5. Reiniciar el ControlPanel
