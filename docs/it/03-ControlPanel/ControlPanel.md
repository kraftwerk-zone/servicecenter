**Download:** [ControlPanel 2025.1 Beta](https://exbzvwxbphbmzoewrupt.supabase.co/storage/v1/object/public/controlpanel/ControlPanel_jre_202510.exe)  
**Dimensione:** 151 MB

### Modifiche
- PremiumSounds di nuovo disponibili, ora anche con numero d'ordine invece di password  
- Infrarosso commutabile, inerzia commutabile  
- Canali e funzioni possono essere nominati tramite le impostazioni di sistema  
- Sequencer preparato  
- Pad virtuale preparato  
- Nuovo firmware per ingressi servo (problema C007)  
- RS1 lampeggia occasionalmente di viola e rifiuta il servizio → risolto  
- Uscite non visualizzate → risolto  
- Scatti dei servi per SBUS → risolto  
- Avviso per utenti SBUS: se non vengono visualizzati “Ingressi PWM RS1 (interni)” → spostare Gas/Data su uscita ricevitore normale, riavviare, ricollegare
