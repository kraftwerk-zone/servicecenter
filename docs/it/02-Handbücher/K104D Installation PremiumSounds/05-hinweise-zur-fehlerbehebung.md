# Indicazioni per la risoluzione dei problemi

- Assicurarsi che la versione del firmware del modulo audio sia aggiornata (almeno 2024.3).
- Verificare la connessione USB tra PC e modulo audio, nel caso in cui il modulo non venga riconosciuto.
- Se il suono non può essere installato, controllare se la licenza è già assegnata a un altro modulo.
- In caso di problemi con l'accesso al Premiumsoundmanager, verificare le credenziali e la connessione a Internet.
