# Avvertenze e consigli

- **Aggiornamento firmware:** Assicurati che il tuo modulo audio sia sempre aggiornato all'ultima versione del firmware per evitare problemi di compatibilità.
- **Installazione multipla:** Se desideri utilizzare un suono su più moduli, verifica prima dell'ordine le fasce di prezzo disponibili.
- **Cambio scheda SD:** Quando sostituisci la scheda SD, il suono deve essere reinstallato come descritto al punto 4.
- **Supporto:** In caso di problemi o domande, il nostro servizio clienti è a tua disposizione telefonicamente o via e-mail.
