# Gamma di funzioni

I tasti possono attivare diverse funzioni a seconda della durata della pressione:

- **Pressione breve:** Simboli arancioni
- **Pressione lunga:** Simboli bianchi
- **Livello aggiuntivo (commutabile con il tasto in alto a sinistra):** Simboli grigi, rispettivamente breve/lungo

## Fila superiore dei tasti

| Pressione breve | Pressione lunga |
| --- | --- |
| Livello pad | Avvio/Arresto motore |
| Freccia sinistra | Servo 1 |
| Luci di emergenza | Modalità festa |
| Freccia destra | Servo 2 |
| Cambio marcia su | Abbagliante lampeggiante |
| Cambio marcia giù | Clacson |

## Fila inferiore dei tasti

| Pressione breve | Pressione lunga |
| --- | --- |
| Luce di posizione | Supporti su |
| Anabbagliante | Supporti giù |
| Abbagliante | Cassone su |
| Fendinebbia/luci posteriori | Cassone giù |
| Luce rotante | Rampa su |
| Accoppiamento rimorchio | Rampa giù |

**Livello pad:** Premendo il tasto in alto a sinistra si passa al livello delle funzioni aggiuntive. Qui possono essere controllate fino a 22 funzioni aggiuntive (undici breve/lungo ciascuna). Le funzioni sono disposte in senso antiorario a partire dal basso a sinistra (simboli grigi).
