# Verifica della regolazione

Aprire l'assistente LiveDati. Premere il pulsante della luce di posizione, il valore percentuale dovrebbe essere circa 86 % (+/- 4 %). Premere il pulsante della frizione a cinghia, il valore dovrebbe essere -86 % (+/- 4 %).
