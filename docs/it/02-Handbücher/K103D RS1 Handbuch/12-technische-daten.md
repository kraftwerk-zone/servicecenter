# Dati tecnici

## Regolatore di velocità

|  |  |
| --- | --- |
| Tensione di esercizio | 6 - 20 V |
| BEC | 6 V / 3 A |
| Frequenza di commutazione | regolabile fino a 32 kHz |
| Funzione frizione | sì |
| Supporto cambio manuale | sì (aggiornamento previsto) |
| Simulazione inerzia | opzionale |
| Cruise control | opzionale (aggiornamento previsto) |
| Funzioni di protezione | sottotensione batteria, cortocircuito motore, sovratemperatura |
| Funzioni servo | funzione proporzionale, commutata, due posizioni, tre posizioni (es. piastra sella, supporti rimorchio, motore cassone/ribaltabile, motore rampa/piattaforma, cambio manuale) |

## Suono

|  |  |
| --- | --- |
| Numero di suoni | illimitato |
| Suoni riproducibili contemporaneamente | ca. 10 |
| Modalità di riproduzione | pressione tasto, ciclo, una volta, permanente, casuale, punti di stop |
| Formati utilizzabili | WAV, MP3 |
| Frequenza di campionamento utilizzabile | tutte (conversione automatica a 44.100 Hz) |
| Regolazione volume | potenziometro, tramite canale, tramite software |
| Scheda SD | max. 32 GB, schede più grandi possono essere formattate in FAT32 da 32 GB |
