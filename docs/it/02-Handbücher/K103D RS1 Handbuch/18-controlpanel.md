## Icone del menu

- **Torna alla panoramica principale**
- **Riconnetti:** Cerca nuovi moduli
- **Ricarica modulo:** Scarta tutte le impostazioni correnti nella vista modulo
- **Apri impostazioni/progetto**
- **Salva impostazioni/progetto:** Sovrascrive le impostazioni caricate dal PC
- **Salva con nome:** Salva con un nuovo nome file
- **Scrivi tutte le impostazioni sul modulo**
