# Aggiungere un nuovo suono

Tipi di suono supportati:

- Sono supportati tutti i formati audio e video comuni.
- Il ControlPanel converte automaticamente nel formato interno necessario, inclusa la bitrate.
- Esempi: WAV, MP3, OGG, MP4, SND

Alla fine della lista dei suoni si trova il pulsante „Nuova voce suono“:

![](images/image\_057.png)

image71.png

Dopo il clic viene creata una nuova riga con il tipo di suono standard dalle impostazioni di base. Con „Apri“ è possibile caricare il file.

In alternativa, i file possono essere trascinati nel ControlPanel tramite drag & drop:

![](images/image\_058.png)

image72.png

  
![](images/image\_059.png)

image73.png

Se un file viene trascinato nella visualizzazione del suono, sostituisce il vecchio suono.

Per i suoni del motore possono essere importati più file contemporaneamente. Se i nomi dei file contengono un numero da tre a cinque cifre, questo viene adottato come „ref RPM“ e le gamme di giri vengono regolate automaticamente. Queste possono essere ottimizzate successivamente nel ControlPanel e caricate sul modulo.
