# Impostazioni

I nostri sistemi EasyBus offrono due tipi di impostazioni:

- **Impostazioni di sistema:** Valide per l'intero modulo o sistema (ad es. intervallo di lampeggio, tipo di ricevitore, tipo di luce freno).
- **Impostazioni di uscita:** Valide solo per la rispettiva uscita (vedi capitolo 7 „Funzioni“).

## Aprire le impostazioni di sistema

Tramite l'icona dell'ingranaggio o il menu „Sistema“ → „Impostazioni di sistema“.

| Nome | Descrizione |
| --- | --- |
| Ritardo di avvio [s] | Ritarda l'avvio del sistema per considerare i tempi di avvio dei radiocomandi. |
| Modalità di apprendimento [attiva] | Attiva/disattiva la modalità di apprendimento manuale tramite leva del gas e dello sterzo. L'apprendimento ControlPanel rimane attivo. |
| Infrarosso [acceso] | Imposta l'uscita 8 in modalità infrarossi. |
| Lampeggio On/Off | Definisce l'intervallo di lampeggio in passi di 10 ms (ad es. 500 ms = 0,5 s). |
| Tipo di ricevitore | Vedi capitolo 8 „Varianti di controllo / Tipo di ricevitore“. |
| Tipo di regolatore di velocità | Vedi capitolo 9.1 „Tipo di regolatore di velocità“. |
| Lampeggio di emergenza in retromarcia | Attivazione automatica del lampeggio di emergenza durante la retromarcia. |
| Sensibilità luce freno | Vedi capitolo 9.2 „Luce freno intelligente“. |
| Durata luce freno | Vedi capitolo 9.2 „Luce freno intelligente“. |
| Tipo di sterzo | Vedi capitolo 9.3 „Tipo di sterzo“. |
| Soglia lampeggio | Vedi capitolo 9.3 „Tipo di sterzo“. |
| Tipo di luce curva | Vedi capitolo 0 „Tipo di luce curva“. |
| Motore parte subito [attivo] | Vedi capitolo 10 „Modulo suono“. |
| Durata avvio motore | Vedi capitolo 10 „Modulo suono“. |
| Pad Tutto On/Off | Configura il comportamento del tasto „Tutto On/Off“ sui pad di controllo. |
| Pad Luce&Suono Abbagliante | Vedi manuale pad di controllo Luce & Suono. |
| Pad Luce&Suono Fendinebbia | Vedi manuale pad di controllo Luce & Suono. |
| Assegnazione K1 – K12 | Vedi capitolo 13.6 „Personalizzato“. |
