# ControlPanel

## Generale

All'avvio del ControlPanel il modello deve essere operativo: batteria carica e collegata, modello e radiocomando accesi e associati. I servi devono poter muoversi.

Il sistema viene rilevato automaticamente e tutti i moduli vengono caricati e visualizzati.

## Panoramica moduli

Cliccando su un modulo si accede alla panoramica delle uscite. Qui è possibile assegnare nomi e impostare le velocità di accensione/spegnimento. I parametri variano a seconda della funzione (vedi capitolo 7 "Funzioni").

Le modifiche vengono salvate con l'icona del floppy disk. Il punto interrogativo mostra testi di aiuto, l'occhio attiva l'uscita sul modello per facilitare le assegnazioni.

## Doppia assegnazione

Ogni uscita può essere assegnata due volte. Esempio:

- Primo livello: luci di posizione & abbagliante
- Secondo livello: lampeggiatori di emergenza

La commutazione tra i livelli può essere configurata.

## Menu

| Funzione | Descrizione |
| --- | --- |
| File | Ricollegare il sistema, ricaricare il modulo, annullare modifiche temporanee, salvare e aprire moduli. |
| Moduli | Visualizzare e caricare tutti i moduli. |
| Impostazioni di sistema | Configurare il comportamento di base del sistema. |
| Informazioni dispositivo | Informazioni sui moduli, versioni software, aggiornamenti, impostazioni di fabbrica. |
| Analisi sistema | Autotest, test comunicazione, individuazione moduli danneggiati. |
| Assistente apprendimento canali | Supporta nell'apprendimento dei canali. |
| Logging | Comunicazione tra ControlPanel e sistema per la ricerca guasti. |
| Assistente dati live | Mostra dati in tempo reale come movimenti degli stick e stati di commutazione. |
| Assistente effetto luci di posizione e KnightRider | Aiuta nella configurazione su più uscite. |
| Impostazioni | Opzioni esperte per impostazioni avanzate (usare con cautela). |
| Info | Informazioni su ControlPanel e report errori. |
