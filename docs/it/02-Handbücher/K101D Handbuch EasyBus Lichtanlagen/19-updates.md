# Aggiornamenti

Dopo la connessione, il ControlPanel offre aggiornamenti disponibili per i moduli bus.

![](images/image\_065.png)

image6.png

**Consigli per l'aggiornamento:**

- Aggiornare sempre un modulo alla volta per evitare errori.
- Evitare interruzioni di corrente durante l'aggiornamento.
- In caso di sussulti o ronzii, scollegare servi/motori.
- Riavviare regolarmente il modello e il ControlPanel, specialmente in caso di ritardi insoliti.
