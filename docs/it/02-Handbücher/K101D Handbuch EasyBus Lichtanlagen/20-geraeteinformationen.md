# Informazioni sul dispositivo

Mostra informazioni di base sui moduli. Con „Mostra dispositivo“ vengono attivate le prime tre uscite, utile in caso di più espansioni universali senza assegnazione di nome.

Con „Reset“ il modulo viene riavviato (simile a scollegare e ricollegare). „Impostazioni di fabbrica“ ripristina le impostazioni di fabbrica, „Aggiorna“ forza un aggiornamento.

Se è disponibile una nuova versione, il ControlPanel lo segnala all’avvio e offre l’aggiornamento. L’aggiornamento manuale è possibile tramite Informazioni sul dispositivo.
