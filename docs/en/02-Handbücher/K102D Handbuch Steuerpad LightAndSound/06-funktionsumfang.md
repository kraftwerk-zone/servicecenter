# Functionality

The buttons can trigger different functions depending on the duration of the press:

- **Short press:** Orange symbols
- **Long press:** White symbols
- **Additional layer (toggle with the top left button):** Gray symbols, each short/long

## Top Row of Buttons

| Short press | Long press |
| --- | --- |
| Pad level | Engine Start/Stop |
| Left indicator | Servo 1 |
| Hazard lights | Party mode |
| Right indicator | Servo 2 |
| Shift up | Flash headlights |
| Shift down | Horn |

## Bottom Row of Buttons

| Short press | Long press |
| --- | --- |
| Parking light | Supports up |
| Low beam | Supports down |
| High beam | Dump bed up |
| Fog light/rear fog light | Dump bed down |
| Rotating beacon | Ramp up |
| Fifth wheel coupling | Ramp down |

**Pad level:** Pressing the top left button switches to the additional function layer. Up to 22 additional functions (eleven short/long each) can be controlled there. The functions are arranged counterclockwise starting from the bottom left (gray symbols).
