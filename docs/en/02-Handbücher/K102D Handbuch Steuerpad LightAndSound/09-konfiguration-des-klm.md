# Configuration of the KLM

![](images/image\_013.png)

image15.png

The control pad can be connected to K1 or K2. Connect the model via USB, turn on the remote control and model, and start the ControlPanel.

Open the system settings via the gear icon or the "System" menu item. Make sure that the KLM is selected in the dropdown list at the top.

Change "Assignment K1" or "Assignment K2" to "Pad Light And Sound".

**Tip:** If layer control is activated for K1, the assignment for K2 is grayed out because this channel is needed for layer switching. If you want to use the pad on K2, select a different entry for K1, e.g., "Custom".

After saving, the KLM is configured for the control pad.

**Tip:** If you are unsure which channel the pad is connected to, the live data view of the ControlPanel can help.
