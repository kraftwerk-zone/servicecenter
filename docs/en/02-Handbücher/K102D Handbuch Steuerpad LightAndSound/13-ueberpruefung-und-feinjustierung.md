# Verification and Fine-Tuning

When the calibration and channels are correctly learned, the pad should work. Test each button and check in the live data whether the corresponding checkmarks are set.

**Tip:** Use the live data view and initially ignore the lights/functions on the model to rule out confusion caused by programming errors.

With cheap remote transmitters, it may be necessary to fine-tune the evaluation of the pad signals.

To do this, press the high beam and then the fog light button. Enter the displayed percentage values in the system settings (gear icon) of the KLM and save them.
