# Checking the Level

Open the Live Data Assistant. Press the parking light button, the percentage value should be about 86% ± 4%. Press the clutch button, the value should be about -86% ± 4%.

If one of the values is above 90%, reduce the level (Setup button and S3 or S2 or S1) and start the learning process again.
