# Commissioning

## Installation in a Carson Transmitter

The control pad must be connected to an analog channel. Switch channels are not suitable. If the pad is connected as shown below, the right stick vertically occupies the control signals of the pad and cannot be used for other purposes.

**Note:** Set the trimmer to one end stop and leave it there. If the deflection later becomes too small in one direction, the trimmer can be moved to the other end stop and the pad relearned.

![](images/image\_005.jpg)

image5.jpeg

Since Carson remote control systems usually do not have a linear control curve, the pad deflections must be set asymmetrically (see section 5.11 "Learning the channels for asymmetric deflections").

The following settings are required:

- Press button 7 and then button 10.
- Press button 6 and then button 2.

![](images/image\_006.jpg)

image6.jpeg

These settings optimize the control range. Then learn the channels accordingly.

Since Carson remote control systems vary greatly, occasionally different settings may be necessary.
