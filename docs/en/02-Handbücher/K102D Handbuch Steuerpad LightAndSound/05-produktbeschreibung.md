# Product Description

The control pad enables the control of all lighting functions as well as the beacon light and the fifth wheel coupling via short tap function (orange symbols). Additionally, functions such as trailer supports, dump bed, ramp, manual transmission, two servos, flash-to-pass, horn, all on/off, and engine start/stop can be controlled by long press (white symbols).

Up to 22 additional functions can be controlled via a layer switch (button top left) (eleven each by short and long press).

The control pad is compatible with all remote control transmitters and occupies only one channel. Ideally, a free expansion channel is used. If no expansion channels are available, an already assigned control channel can also be used, but it will no longer be available for other applications.

Signal evaluation takes place in the central control units KLM 4/0, KLM 4/12 – 500, or KLM 4/16. Older KLM light assistants are also compatible with the current ControlPanel after an update and can be used without restrictions.

Commissioning takes place in five steps, which are explained in detail below:

1. Installation in the remote control transmitter  
2. Configuration of the KLM  
3. Setting the output level  
4. Learning the channels  
5. Optional: Fine adjustment  

Note: Steps 3 and 4 may need to be repeated several times until the optimal output level is found.
