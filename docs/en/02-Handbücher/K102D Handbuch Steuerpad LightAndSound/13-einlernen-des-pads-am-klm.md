# Learning the Pad on the KLM

There are different learning methods. The easiest is via the LiveData in the ControlPanel:

![](images/image\_014.png)

image14.png

1. Click the Learn button. The text changes to "Center position".
2. Click again to confirm the center position.
3. The text changes to "Top / left". Hold down the layer button and click again.
4. The text changes to "Bottom / right". Hold down the minus button and click again.
5. The text shows "Save". Click one last time to save the values.

Alternatively, you can use the "Learn Channels" assistant in the ControlPanel, which guides you through the entire learning process. A correct assignment of K1/K2 in the system settings is required.

A standard variant without the ControlPanel is also possible. Details can be found in the manual of the EasyBus lighting systems or the RS1.

When learning the pad, first press the layer button, then the minus button.
