# Verification and Fine-Tuning

If the calibration and channels are correctly learned, the pad should already work. Test all buttons one after the other and check in the live data whether the corresponding functions are activated.

**Tip:** Use the live data view and initially ignore the model functions to exclude confusion caused by possible programming errors.

With inexpensive remote control transmitters, it may be necessary to fine-tune the evaluation of the pad signals, as the channels are not always controlled proportionally.
