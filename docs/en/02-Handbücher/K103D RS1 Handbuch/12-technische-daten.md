# Technical Data

## Speed Controller

|  |  |
| --- | --- |
| Operating Voltage | 6 - 20 V |
| BEC | 6 V / 3 A |
| Switching Frequency | adjustable up to 32 kHz |
| Clutch Function | yes |
| Manual Transmission Support | yes (update planned) |
| Inertia Simulation | optional |
| Cruise Control | optional (update planned) |
| Protection Functions | battery undervoltage, motor short circuit, overtemperature |
| Servo Functions | proportional function, switched, two positions, three positions (e.g. fifth wheel plate, trailer supports, motor dump/tilt, motor ramp/platform, manual transmission) |

## Sound

|  |  |
| --- | --- |
| Number of Sounds | unlimited |
| Simultaneously Playable Sounds | approx. 10 |
| Playback Modes | button press, loop, once, permanent, random, stop points |
| Supported Formats | WAV, MP3 |
| Supported Sampling Rate | all (automatic conversion to 44,100 Hz) |
| Volume Control | potentiometer, via channel, via software |
| SD Card | max. 32 GB, larger cards can be formatted to 32 GB FAT32 |
