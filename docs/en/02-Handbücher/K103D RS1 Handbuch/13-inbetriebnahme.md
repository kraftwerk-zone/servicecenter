# Commissioning

The RS1 must be connected to five channels of the receiver. The use of a control pad for light and sound can be configured via a jumper.

## Control pad on K3 (right jumper set)

- Motor brake is active as long as K2 is pressed upwards (if supported by the model sound).
- If K2 is pulled down and then tapped to the right, the horn switches to horn and fanfare (if supported).

## Control without control pad (upper jumper not set)

The channel assignment is denser here. Throttle, steering, volume, and load are controlled as when using the control pad.

- Motor brake as a memory function: short tap activates/deactivates the motor brake.
- Long press upwards starts/stops the motor.
- Half tap activates the horn.
- If K2 is pulled down and then tapped to the right, the horn switches to horn and fanfare.

## Note on using a push button on K3

If a push button is used for K3, only full functions can be switched. With appropriate mixers of the remote control, half functions can also be realized.
