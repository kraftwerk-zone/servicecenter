# Connecting to the Receiver

![](images/image\_016.png)

image8.png

Before connecting, check the channel assignment of your receiver. Use a servo to test each output for correct function. Note the assignment.

![](images/image\_015.jpg)

image22.jpeg

**Important:** Block the model to prevent unintended movement. Then connect all servo cables to the corresponding receiver outputs.

When using IBUS, SBUS, or SUMD, the RS1 switches automatically. The channel assignment can be adjusted in the live data if needed.
