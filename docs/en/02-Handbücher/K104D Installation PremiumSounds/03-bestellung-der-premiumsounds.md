# Ordering Premium Sounds

Premium sounds can be conveniently ordered through our online shop. In the **"Sounds"** tab, you will find the current selection of available premium sounds.

Important: Each sound can only be installed on a single sound module. If you want to use the same sound on multiple modules, we offer attractive pricing tiers to help you save costs.
