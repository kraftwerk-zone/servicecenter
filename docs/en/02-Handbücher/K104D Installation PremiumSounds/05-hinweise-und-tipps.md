# Notes and Tips

- **Firmware Update:** Make sure your sound module is always operated with the latest firmware version to avoid compatibility issues.
- **Multiple Installation:** If you want to use a sound on multiple modules, check the available price tiers before ordering.
- **Changing the SD Card:** When replacing the SD card, the sound must be installed again as described in step 4.
- **Support:** If you have any problems or questions, our customer service is available by phone or email.
