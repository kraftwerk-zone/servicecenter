# Device Information

Displays basic information about modules. Using "Show Device" turns on the first three outputs, useful for multiple universal expansions without naming.

"Reset" restarts the module (similar to unplugging and plugging back in). "Factory Settings" resets to factory defaults, "Update" forces an update.

If a new version is available, the ControlPanel indicates this at startup and offers the update. Manual updates are possible via Device Information.
