# Troubleshooting

## General

- **Turn signals and cornering lights respond reversed:** Channels are not learned.
- **Reverse light turns on when driving forward:** Channels are not learned.
- **Steering servo turns incorrectly:** Adjust direction after learning with ControlPanel. Changes in the transmitter after learning neutralize the setting.
- **Light functions cannot be controlled:** Check if the model is operational (battery charged, transmitter and model switched on). Check the wiring. Test the speed controller directly on the receiver without the light assistant.
- **Lighting system shows no reaction:** The green LED of the light assistant should blink twice briefly after switching on. Channels must be learned.
- **Learning the channels not possible:** Check connection and function of the receiver outputs with a servo. Open LiveData Assistant and check if each channel responds cleanly. Start the learning assistant and follow the instructions.
- **No connection to PC:** Driver installed? Model switched on and connected via USB? COM port correctly selected?
- **Module not found after update:** Close ControlPanel, restart model (disconnect battery, wait briefly, reconnect). Start ControlPanel and check output. If "Damaged module" appears, perform update. In case of problems, disconnect all modules except the affected one and repeat update.
- **No damaged module displayed:** Disconnect all modules from the bus, close ControlPanel, restart model, start ControlPanel, connect and connect the problematic module. Perform update or contact support.
