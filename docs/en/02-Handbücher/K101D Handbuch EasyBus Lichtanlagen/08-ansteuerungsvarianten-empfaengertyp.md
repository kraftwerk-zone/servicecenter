# Control Variants / Receiver Type

The control type can be selected in the system settings of the KLM 4/16 under "Receiver Type".

## Normal 4xPWM

The KLM 4/16 is connected to up to four channels of the receiver. The channels must be manually or with the ControlPanel trained to define center position, end points, and rotation directions.

## CPPM

If your receiver offers a CPPM output (summary signal), it can be connected to the CPPM input of the KLM. If necessary, activate the CPPM output on the receiver (see the remote control system manual).

The channel assignment is done via the LiveData view by moving the sticks one after another and assigning the corresponding channels.

After the assignment, the channels must be trained as usual.

## I-BUS

For receivers with I-BUS output (e.g. Turnigy), the same procedure applies as for CPPM.

## S-BUS

For receivers with S-BUS output (e.g. FUTABA), the same procedure applies as for CPPM.
