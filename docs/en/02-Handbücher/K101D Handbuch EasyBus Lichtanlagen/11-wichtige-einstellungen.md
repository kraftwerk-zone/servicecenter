# Important Settings

## Engine Starts Immediately

Depending on the sound module, the system setting "Engine starts immediately [active]" must be adjusted:

- Sound module starts engine immediately: check the box.
- Engine must be started manually: uncheck the box.

**Caution:** The drive motor is active in any case!

## Lights Flicker

Each lighting board and switch extension (KLB) has system settings for "Start effect" (flickering at start) and "Motor stop light reduction" (dimming when motor stops). Use the latter option only if no components with integrated electronics are connected.

The duration of the flickering can be adjusted with "Engine start duration" to match the start sound duration.
