# Control Variants / Receiver Types

The control type can be selected in the system settings of the KLM 4/16 under "Receiver Type".

## Normal 4xPWM

The KLM 4/16 is connected to up to four channels of the receiver. The channels must be manually or via the ControlPanel learned to define center positions, end points, and rotation directions.

## CPPM

If your receiver offers a CPPM output (summary signal), it can be connected to the CPPM input of the KLM. Activate the CPPM output on the receiver if necessary (see the remote control system manual). The channel assignment is done via the live data view in the ControlPanel.

## I-BUS

For receivers with an I-BUS output (e.g., Turnigy), this can be connected to the I-BUS input of the KLM. Activation on the receiver is required. The further steps correspond to those of CPPM.

## S-BUS

For receivers with an S-BUS output (e.g., FUTABA), the same procedure applies as with CPPM and I-BUS.
