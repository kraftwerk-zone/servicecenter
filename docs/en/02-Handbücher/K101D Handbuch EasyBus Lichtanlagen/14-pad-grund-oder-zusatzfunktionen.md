# Pad Basic or Additional Functions

Pads encode key presses into two upward and downward movements. The KLM decodes these and controls the functions accordingly.

The pad is connected to a control device in the remote control transmitter, the channel to K1 or K2 of the KLM.

To learn the channels, the pad must be set to setup mode (press both setup buttons simultaneously). Alternatively, the "Learn Channels" assistant in the ControlPanel is recommended.
