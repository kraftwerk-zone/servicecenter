## Servo Functions

| Function | Description |
| --- | --- |
| Proportional | Control of a servo or motor controller with a proportional channel. |
| Proportional Switched | Control with proportional channel, active only with a certain switch function (e.g., disable steering when lift axle is raised). |
| Two Positions | Servo moves from position 1 to position 2 when the switch function is activated. |
| Three Positions | Servo moves from position 2 to position 1 or 3 depending on the activated switch function. |
| Fifth Wheel Plate | Control of the fifth wheel plate (manual gearbox). |

## Motor Functions

| Function | Description |
| --- | --- |
| Proportional | Control of a motor with a proportional channel. |
| Proportional Switched | Motor control only when the switch function is activated. |
| Drop Legs | Motor control when selecting the drop legs function. |
| Dump Bed | Motor control when selecting the dump bed function. |
| Tipper | Motor control when selecting the tipping function. |
