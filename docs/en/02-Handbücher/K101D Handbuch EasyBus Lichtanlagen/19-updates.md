# Updates

After connecting, the ControlPanel offers available updates for bus modules.

![](images/image\_065.png)

image6.png

**Update Tips:**

- Always update only one module at a time to avoid errors.
- Avoid power interruptions during the update.
- Disconnect servos/motors if twitching or humming occurs.
- Restart the model and ControlPanel regularly, especially if there are unusual delays.
