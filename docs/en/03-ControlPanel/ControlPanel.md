## Beta2025.1 – ControlPanel (07/08/2025)
**Download:** https://kraftwerk-zone.com/_downloads/ControlPanel_jre_202510.exe  
**Size:** 151 MB

### Changes
- PremiumSounds available again, now also with order number instead of password
- Infrared switchable, inertia switchable
- Channels and functions can be named via system settings
- Sequencer prepared
- Virtual pad prepared
- New firmware for servo inputs (C007 issue)
- RS1 occasionally blinks violet and refuses service → fixed
- Outputs were not displayed → fixed
- Twitching of servos for SBUS → fixed
- Note for SBUS users: If no “RS1 PWM inputs (internal)” are displayed → switch throttle/data to normal receiver output, restart, reconnect
