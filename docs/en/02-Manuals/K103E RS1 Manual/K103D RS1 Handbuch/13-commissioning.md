# Commissioning

The RS1 must be connected to five channels of the receiver. Using jumpers, the use of a control pad for lighting and sound can be configured.

![](images/image\_004.png)

image9.png

When the right jumper is set, the control pad is connected to channel 3 (K3). The engine brake remains active as long as channel 2 (K2) is pushed up, provided the model sound supports this. If K2 is pushed down and then tapped right, the horn switches to horn and fanfare sounds, if supported by the model sound.

![](images/image\_014.png)

If the upper jumper is not set, control is via buttons on K3. Without a control pad, channel assignments are denser. Throttle, steering, volume, and load operate as with the control pad. The engine brake functions as a memory toggle: a short tap activates it, another tap deactivates it. Holding the button up starts/stops the motor. Half taps activate the horn. Pushing K2 down and then tapping right switches the horn to horn and fanfare, if supported.
