# Adding New Sounds

Supported sound types include all common audio and video formats. The ControlPanel automatically converts files to the required format and bitrate.

Examples: WAV, MP3, OGG, MP4, SND.

To add a new sound, click the "New Sound Entry" button at the bottom:

![](images/image\_054.png)

image71.png

A new row is created with the default sound type from basic settings. Use the "Open" button to load files.

Alternatively, drag and drop files into the ControlPanel:

![](images/image\_055.png)

image72.png

![](images/image\_056.png)

image73.png

Dragging a file into the sound visualization area replaces the existing sound instead of adding a new one.

For motor sounds, you can preselect the default type (e.g., Motor Load) and drag multiple files simultaneously. If filenames contain a 3-5 digit number, it will be used as the reference RPM, and the RPM ranges will be set accordingly. These can be fine-tuned later in ControlPanel before uploading to the module.
