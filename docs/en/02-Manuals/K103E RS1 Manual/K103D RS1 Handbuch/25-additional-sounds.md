## Trigger Settings

- Trigger Types: Rising edge, falling edge, both edges.
- Playback Types: Loop, always play, return to start or continue if stopped.
- All combinations are possible.

Available triggers depend on whether a control pad or SUMD remote is used. Functions correspond to those in Live Data.

## Stop Points

Stop points can be defined in seconds for each sound. If "always play" is active, the sound plays until the next stop point or continues if the function remains active.

![](images/image\_052.png)

image69.png

## Random Playback

Sounds can be set to start randomly every x to y seconds while active.
