# Connecting to the Receiver

![](images/image\_013.png)

image8.png

Before connecting the RS1 to the receiver, verify your receiver's channel assignments. Use a servo to test each output for correct function and note the assignments.

![](images/image\_012.jpg)

image22.jpeg

**Warning:** It is recommended to elevate the model to prevent unintended movement during setup. Then connect all servo cables to the corresponding receiver outputs. If using an IBUS, SBUS, or SUMD receiver, the RS1 will automatically adjust. Channel assignments can be modified in Live Data if necessary.
