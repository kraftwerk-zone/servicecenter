# Technical Specifications

## Speed Controller

| Parameter | Specification |
| --- | --- |
| Operating Voltage | 6 - 20 V |
| BEC Output | 6 V / 3 A |
| Switching Frequency | Adjustable up to 32 kHz |
| Clutch Function | Yes |
| Gear Shift Support | Yes (future update) |
| Inertia Simulation | Optional |
| Cruise Control | Optional |
| Protection Features | Battery undervoltage, motor short circuit, overtemperature |
| Servo Functions | Proportional control, two-position, three-position, saddle plate, trailer supports, motor for dump truck, ramp/platform motor, gear shift motor |

## Sound System

| Parameter | Specification |
| --- | --- |
| Number of Sounds | Unlimited |
| Simultaneous Playback | Approx. 10 |
| Playback Modes | Key, Loop, Once, Permanent, Random, Stop Points |
| Supported Formats | WAV, MP3 |
| Sampling Rate | All (automatic conversion to 44,100 Hz) |
| Volume Control | Potentiometer, channel control, software |
| SD Card | Up to 32 GB FAT32; larger cards can be formatted to 32 GB |
