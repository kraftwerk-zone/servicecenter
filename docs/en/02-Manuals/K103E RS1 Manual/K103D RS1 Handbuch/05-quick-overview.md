# Quick Overview

## Software Version

Always download the latest version of the ControlPanel software and connect your module. All available updates, including new features and bug fixes, will be installed automatically.

## Status LED Indicators

- **Normal operation:** LED blinks green briefly every two seconds.
- **Standby mode:** LED blinks yellow slowly; no control signals are accepted.
- **Fast red blinking:** Indicates undervoltage; warning lights activate and the motor shuts down.
- **Slow red or other colors:** Please contact support.

## Standby Switch

The switch allows you to turn the model on/off or set it to Standby mode. In Standby, all functions continue running, but no control signals from the remote are accepted, effectively creating a demo mode.

In the OFF position, the controller and all connected servos and bus modules are switched off.

**Warning:** A small current still flows in OFF mode, which can lead to battery deep discharge if the model is stored in this state. Always disconnect the battery, at least at the end of the day.
