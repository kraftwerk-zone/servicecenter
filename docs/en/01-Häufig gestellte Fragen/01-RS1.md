# Frequently Asked Questions - RS1

---

## RS1 gets stuck in a startup loop.

If the RS1 restarts continuously in a loop, it is usually due to very old firmware combined with a defective SD card.

**Solution steps**

1. Replace the SD card (max. 32 GB, FAT32 recommended)  
2. Install the latest version of the ControlPanel  
3. Copy contents from  
   `Dokumente\KraftwerK\ControlPanel\sdcard_original`  
   to the new SD card  
4. Restart the RS1  
5. Restart the ControlPanel  
6. Perform all available updates  

---

## RS1 aborts updates – why?

If an update does not complete, the firmware directory is often missing on the SD card.

**How to fix the problem**

1. Open the SD card directly on the PC  
   or in the ControlPanel under  
   **Sound Settings → Open SD Card**  
2. Check if a folder named **Firmware** exists on the SD card  
3. If not present: create a new folder named **Firmware**  
4. Restart the RS1  
5. Restart the ControlPanel
