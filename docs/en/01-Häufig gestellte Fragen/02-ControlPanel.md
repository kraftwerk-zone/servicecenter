# Frequently Asked Questions – ControlPanel
by Wolfgang · {{ git_revision_date_localized }}

---

## How do I install the ControlPanel?

In the kraftwerk.shop Service Center you will find the current  
[Downloads](https://kraftwerk.shop/Servicecenter/03-ControlPanel/ControlPanel/).

1. Download the file  
2. Run it  
3. If Windows shows a security warning: choose **"Run anyway"**  
   (the link is sometimes a bit hidden)  
4. Complete the installation as usual  

After installation, the ControlPanel starts automatically and usually detects connected modules immediately.

---

## The installation starts every time for me.

Please **do not** start the ControlPanel via the downloaded file.  
This file is the **installer** – and therefore runs the installation again every time.

Instead, open the ControlPanel via:

- the **Start menu**, or  
- the **desktop shortcut**

---

## The system settings are not displayed.

On some computers, display issues may occur.  
In many cases, simply **resizing the window** helps, as this causes the interface to be rebuilt.
