# Funkce

## Světelné funkce

| Funkce | Popis |
| --- | --- |
| Levý blinkr / Pravý blinkr | S automatickým návratem, volitelně automaticky při natočení volantu (viz sekce 9.3). Frekvence blikání nastavitelná v systémových nastaveních. |
| Výstražné světlo | Automaticky aktivovatelné při couvání. |
| Parkovací světlo | ![](images/image\_003.png)  image9.png |
| Parkovací světlo & potkávací světlo | Volitelně s xenonovým efektem pro potkávací světlo. |
| Parkovací světlo & dálkové světlo | Volitelně s xenonovým efektem pro dálkové světlo. |
| Parkovací světlo, potkávací & dálkové světlo | Volitelně s xenonovým efektem pro potkávací nebo dálkové světlo. |
| Potkávací světlo | Volitelně s xenonovým efektem. |
| Potkávací světlo, dálkové světlo | Volitelně s xenonovým efektem pro potkávací nebo dálkové světlo. |
| Dálkové světlo | Volitelně s xenonovým efektem. |
| Mlhové světlo | Expertní funkce: přední světlo se všemi kombinacemi možné. |
| Denní svícení | Vždy zapnuto, může být ztlumeno, pokud jsou zapnuté blinkry a/nebo parkovací světlo. |
| Odbočovací světlo / světlo do zatáček | Aktivní při zapnutí blinkru nebo řízení přes prahovou hodnotu (viz systémová nastavení). Volitelně s mlhovým světlem. |
| Brzdové světlo | Inteligentní řízení s nastavitelnou citlivostí a dobou dohoření. |
| Couvací světlo | Jednoduchý regulátor vpřed/vzad nebo s oddělenou brzdou/couváním. |
| Zadní světlo & brzdové světlo / zadní mlhové světlo | Všechny kombinace možné. |
| Americký blinkr | Parkovací světlo a blinkr přes jednu LED. |
| Americké zadní světlo | Parkovací světlo, blinkr a brzdové světlo přes jednu LED. |
| Zapnutí/vypnutí výstražného světla | Přepíná výstup na 100 %, když je výstražné světlo aktivní. |
| Jednoduchý přepínač / dvojitý přepínač | Přepíná výstupy podle zvolené funkce a jasu. |
| Multiblikr | Expertní funkce s nastavitelným zpožděním startu, počtem pulzů a pauzami. |
| Pořadové světlo | Konfigurace pořadového světla přes několik výstupů. |
| KnightRider | Známý efekt pořadového světla s nastavitelným jasem a rychlostí. |

## Servofunkce

| Funkce | Popis |
| --- | --- |
| Proporcionální | Řízení serva nebo motorového regulátoru s proporcionálním kanálem. |
| Proporcionální spínané | Řízení s proporcionálním kanálem, aktivní pouze při určité spínací funkci. |
| Dvě polohy | Servo se pohybuje mezi dvěma polohami podle spínací funkce. |
| Tři polohy | Servo se pohybuje mezi třemi polohami, řízeno dvěma spínacími funkcemi. |
| Sedlová deska / řazení | Speciálně pro modelové funkce jako sedlová deska nebo řazení. |

## Motorové funkce

| Funkce | Popis |
| --- | --- |
| Proporcionální | Řízení motoru s proporcionálním kanálem. |
| Proporcionální spínané | Řízení motoru pouze při aktivní spínací funkci. |
| Sedlové podpěry / korba / sklápěč | Řízení motoru pro specifické modelové funkce. |
