# Rozsah funkcí

Tlačítka mohou podle délky stisku vyvolat různé funkce:

- **Krátký stisk:** Oranžové symboly
- **Dlouhý stisk:** Bílé symboly
- **Dodatečná úroveň:** Přepnutí levým horním tlačítkem, umožňuje dalších 22 funkcí (šedé symboly)

## Horní řada

| Krátký stisk | Dlouhý stisk |
| --- | --- |
| Úroveň panelu | Start/Stop motoru |
| Levý blinkr | Servo 1 |
| Výstražné světlo | Party režim |
| Pravý blinkr | Servo 2 |
| Zařadit vyšší | Dálkové světlo |
| Zařadit nižší | Houkačka |

## Dolní řada

| Krátký stisk | Dlouhý stisk |
| --- | --- |
| Parkovací světlo | Zvednutí podpěr |
| Potkávací světlo | Složení podpěr |
| Dálkové světlo | Zvednutí korby |
| Mlhové světlo/zadní mlhovka | Složení korby |
| Výstražné světlo | Zvednutí rampy |
| Tažné zařízení | Složení rampy |

**Dodatečná úroveň:** Stisknutím levého horního tlačítka se aktivuje úroveň dodatečných funkcí. Začínaje zleva dole a proti směru hodinových ručiček lze ovládat až 22 dodatečných funkcí (každá krátký/dlouhý stisk po jedenácti).
