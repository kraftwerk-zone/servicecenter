# Funkčnost

Tlačítka mohou podle délky stisku vyvolat různé funkce:

- **Krátký stisk:** Oranžové symboly
- **Dlouhý stisk:** Bílé symboly
- **Dodatečná úroveň (přepínatelná levým horním tlačítkem):** Šedé symboly, vždy krátký/dlouhý

## Horní řada tlačítek

| Krátký stisk | Dlouhý stisk |
| --- | --- |
| Úroveň panelu | Start/Stop motoru |
| Levý blinkr | Servo 1 |
| Výstražné světla | Párty režim |
| Pravý blinkr | Servo 2 |
| Zařadit vyšší rychlost | Dálkové světlo |
| Zařadit nižší rychlost | Klakson |

## Dolní řada tlačítek

| Krátký stisk | Dlouhý stisk |
| --- | --- |
| Parkovací světlo | Zvednutí podpěr |
| Potkávací světlo | Složení podpěr |
| Dálkové světlo | Zvednutí korby |
| Mlhové světlo / zadní světlo | Složení korby |
| Výstražné světlo | Zvednutí rampy |
| Tažné zařízení | Složení rampy |

**Úroveň panelu:** Stisknutím levého horního tlačítka se přepne na úroveň dodatečných funkcí. Zde lze ovládat až 22 dodatečných funkcí (každou krátkým/dlouhým stiskem jedenáct). Funkce jsou uspořádány proti směru hodinových ručiček, počínaje z levého dolního rohu (šedé symboly).
