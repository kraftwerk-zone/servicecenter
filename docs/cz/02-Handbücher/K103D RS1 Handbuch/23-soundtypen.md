# Typy zvuků

![](images/image\_045.png)

image59.png

- **Standardní zvuk motoru:** Zrychlení a jízda plynule
- **Zvuk motoru při zatížení:** Zrychlení a jízda v režimu zatížení
- **Zvuk motoru při zpomalení:** Vozidlo zpomaluje
- **Zvuk motorové brzdy:** Vozidlo zpomaluje s motorovou brzdou
- **Zvuk změny zatížení:** Jednorázový zvuk při změně jízdního stavu
- **Start/Stop motoru:** Zvuk startu a zastavení, spínatelný podle zatížení
- **Zvuk chybného startu:** Zvuk při chybném startu
- **Turbo:** Pískání turba
- **Zvuk řazení:** Zvuk při změně rychlostního stupně
- **Vzduchová brzda zapnuto/vypnuto:** Zvuky vzduchové brzdy
- **Zpětný varovný signál:** Při couvání
- **Klakson/trubka/fanfára:** Přepínatelné přes klakson
- **Doplňkový zvuk:** Uživatelsky definované zvuky
- **Zvuk zapnutí:** Po úplném spuštění modulu
- **Zvuk rozjezdu:** Při rozjezdu (většinou není potřeba)
- **Zvuk zastavení:** Při zastavení
- **MP3:** Lze přehrát pouze jedno MP3 najednou
