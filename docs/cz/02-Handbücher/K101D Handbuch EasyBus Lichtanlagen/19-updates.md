# Aktualizace

Po připojení nabízí ControlPanel dostupné aktualizace pro autobusové moduly.

![](images/image\_065.png)

image6.png

**Tipy pro aktualizaci:**

- Vždy aktualizujte pouze jeden modul najednou, aby se předešlo chybám.
- Vyhněte se přerušení napájení během aktualizace.
- Při škubání nebo brumu odpojte serva/motory.
- Pravidelně restartujte model a ControlPanel, zejména při neobvyklých prodlevách.
