# Informace o zařízení

Zobrazuje základní informace o modulech. Pomocí „Zobrazit zařízení“ se zapnou první tři výstupy, což je praktické u více univerzálních rozšíření bez pojmenování.

Pomocí „Reset“ se modul restartuje (srovnatelné s odpojením a připojením). „Výchozí nastavení“ obnoví tovární nastavení, „Aktualizace“ vynutí aktualizaci.

Pokud je k dispozici nová verze, ControlPanel na to při spuštění upozorní a nabídne aktualizaci. Manuální aktualizace je možná přes informace o zařízení.
