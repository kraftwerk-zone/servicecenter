## Servofunkce

| Funkce | Popis |
| --- | --- |
| Proporcionální | Řízení serva nebo regulátoru motoru s proporcionálním kanálem. |
| Proporcionální spínané | Řízení s proporcionálním kanálem, aktivní pouze při určité spínací funkci (např. vypnutí řízení při zvednutí nápravy). |
| Dvě polohy | Servo se pohybuje z pozice 1 do pozice 2 při aktivaci spínací funkce. |
| Tři polohy | Servo se pohybuje z pozice 2 do pozice 1 nebo 3 podle aktivované spínací funkce. |
| Deska sedla | Řízení desky sedla (převodovka). |
