# Nastavení

Naše systémy EasyBus nabízejí dva typy nastavení:

- **Systémová nastavení:** Platí pro celý modul nebo systém (např. interval blikání, typ přijímače, typ brzdového světla).
- **Nastavení výstupu:** Platí pouze pro daný výstup (viz kapitola 7 „Funkce“).

## Otevření systémových nastavení

Pomocí ikony ozubeného kola nebo menu „Systém“ → „Systémová nastavení“.
