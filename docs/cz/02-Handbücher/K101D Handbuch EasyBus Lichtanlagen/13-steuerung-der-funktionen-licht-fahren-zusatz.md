# Řízení funkcí Světlo – Jízda – Doplňky

Standardní řízení probíhá přes čtyři proporcionální kanály (plyn vlevo, řízení vpravo). Kanál K2 přepíná mezi skupinami Světlo, Jízda a Doplňkové funkce.

## Naučení kanálů

1. Plyn – brzda  
2. Vlevo – vpravo  
3. K1 vlevo – vpravo  
4. K2 nahoru – dolů

## Přepínání světelných funkcí

- Stálé světlo, potkávací světlo: K2 nahoru + krátce ťuknout K1 vlevo  
- Dálkové světlo: K2 nahoru + krátce ťuknout K1 vpravo (pouze pokud je stálé světlo zapnuto)  
- Mlhové světlomety, zadní mlhové světlo: K2 nahoru + dlouze ťuknout K1 vpravo (pouze pokud je stálé světlo zapnuto)  
- Blinkry vlevo/vpravo: K2 uprostřed + krátce ťuknout K1 vlevo/vpravo  
- Výstražné blinkry: K2 uprostřed + dlouze ťuknout K1 vpravo  
- Světelný signál: K2 uprostřed + dlouze ťuknout K1 vlevo (poté K1 vlevo pro zapnutí/vypnutí)  
- Doplňkové funkce (plné/poloviční, krátké/dlouhé): K2 dolů + odpovídající pohyb K1
