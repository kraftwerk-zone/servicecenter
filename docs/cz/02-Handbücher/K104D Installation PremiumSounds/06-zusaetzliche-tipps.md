# Další tipy

- Uchovávejte svá uživatelská data v bezpečí, abyste měli kdykoli přístup ke svým zakoupeným zvukům.
- Využijte cenové hladiny, pokud chcete vybavit více modulů stejnými zvuky, abyste ušetřili náklady.
- Při výměně SD karty nebo při změně modulu lze zvuky bez problémů znovu nainstalovat, pokud je licence platná.
