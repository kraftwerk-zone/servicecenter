# Pokyny pro odstraňování chyb

- Ujistěte se, že verze firmwaru zvukového modulu je aktuální (alespoň 2024.3).
- Zkontrolujte USB připojení mezi PC a zvukovým modulem, pokud modul není rozpoznán.
- Pokud nelze zvuk nainstalovat, ověřte, zda licence již není přiřazena jinému modulu.
- Při problémech s přihlášením do Premiumsoundmanageru zkontrolujte své přihlašovací údaje a internetové připojení.
