# Centrální časově řízené funkce

Centrální časově řízené funkce, jako jsou směrová světla, jsou řízeny dvěma digitálními funkcemi:

- Informace, zda je funkce obecně zapnuta nebo vypnuta
- Impuls, který zapíná a vypíná světlo

Pro řízení směrových světel stačí nastavit impulsní informaci. Druhá informace může být použita pro mixéry a další přepínače, například pro přepnutí na druhou konfiguraci při zapnutých směrových světlech.
