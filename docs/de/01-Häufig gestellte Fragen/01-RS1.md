# Häufig gestellte Fragen - RS1

---

## RS1 hängt beim Starten in einer Schleife.

Wenn sich der RS1 in einer Dauerschleife neu startet, liegt meist eine sehr alte Firmware in Kombination mit einer defekten SD‑Karte vor.

**Lösungsschritte**

1. SD‑Karte austauschen (max. 32 GB, FAT32 empfohlen)  
2. Neueste Version des ControlPanels installieren  
3. Inhalte aus  
   `Dokumente\KraftwerK\ControlPanel\sdcard_original`  
   auf die neue SD‑Karte kopieren  
4. RS1 neu starten  
5. ControlPanel neu starten  
6. Alle verfügbaren Updates durchführen  

---

## RS1 bricht Updates ab – warum?

Wenn ein Update nicht durchläuft, fehlt häufig das Firmware‑Verzeichnis auf der SD‑Karte.

**So behebst du das Problem**

1. SD‑Karte direkt am PC öffnen  
   oder im ControlPanel unter  
   **Soundeinstellungen → SD‑Karte öffnen**  
2. Prüfen, ob auf der SD‑Karte ein Ordner **Firmware** existiert  
3. Falls nicht vorhanden: Ordner **Firmware** neu anlegen  
4. RS1 neu starten  
5. ControlPanel neu starten
