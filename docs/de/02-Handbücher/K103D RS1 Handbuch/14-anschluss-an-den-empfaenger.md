# Anschluss an den Empfänger

![](images/image\_016.png)

image8.png

Überprüfen Sie vor dem Anschluss die Kanalbelegung Ihres Empfängers. Verwenden Sie dazu ein Servo und testen Sie jeden Ausgang auf korrekte Funktion. Notieren Sie die Belegung.

![](images/image\_015.jpg)

image22.jpeg

**Wichtig:** Bocken Sie das Modell auf, um ein unbeabsichtigtes Losfahren zu verhindern. Schließen Sie anschließend alle Servokabel an die entsprechenden Empfängerausgänge an.

Bei Verwendung von IBUS, SBUS oder SUMD stellt sich der RS1 automatisch um. Die Kanalzuordnung kann bei Bedarf in den LiveDaten angepasst werden.
