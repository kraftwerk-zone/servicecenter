# Einlernen des Pads am KLM

Es gibt mehrere Varianten, das Pad einzulernen. Die einfachste ist über die LiveDaten im ControlPanel:

![](images/image\_014.png)

image14.png

1. Klicken Sie auf den Einlernen-Button, der Text ändert sich auf „Mittelstellung“.
2. Klicken Sie erneut, um die Mittelstellung zu bestätigen.
3. Drücken Sie die Ebenentaste und klicken Sie erneut („Oben / links“).
4. Drücken Sie die Minustaste und klicken Sie erneut („Unten / rechts“).
5. Zum Schluss klicken Sie auf „Speichern“.

Alternativ können Sie den „Kanäle Einlernen“-Assistenten im ControlPanel verwenden, der Sie durch das Einlernen aller Kanäle führt. Wenn die Belegung K1/K2 korrekt gesetzt ist, wird das Pad automatisch berücksichtigt.

Eine Standardvariante ohne ControlPanel ist ebenfalls möglich. Details hierzu finden Sie im Handbuch für die EasyBus Lichtanlagen oder des RS1.

Beim Einlernen des Pads drücken Sie zuerst die Ebenentaste, dann die Minustaste.
