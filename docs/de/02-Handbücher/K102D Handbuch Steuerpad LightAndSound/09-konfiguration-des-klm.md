# Konfiguration des KLM

![](images/image\_013.png)

image15.png

Das Steuerpad kann an K1 oder K2 angeschlossen werden. Verbinden Sie das Modell per USB, schalten Sie Fernsteuerung und Modell ein und starten Sie das ControlPanel.

Öffnen Sie die Systemeinstellungen über das Zahnradsymbol oder den Menüpunkt „System“. Achten Sie darauf, dass in der Auswahlliste oben der KLM ausgewählt ist.

Ändern Sie „Belegung K1“ oder „Belegung K2“ auf „Pad Light And Sound“.

**Tipp:** Ist für K1 die Ebenensteuerung aktiviert, ist die Belegung für K2 ausgegraut, da dieser Kanal für die Ebenenumschaltung benötigt wird. Möchten Sie das Pad an K2 verwenden, wählen Sie für K1 einen anderen Eintrag, z. B. „Benutzerdefiniert“.

Nach dem Speichern ist der KLM für das Steuerpad konfiguriert.

**Tipp:** Wenn Sie unsicher sind, an welchem Kanal das Pad angeschlossen ist, hilft die LiveDaten-Ansicht des ControlPanels.
