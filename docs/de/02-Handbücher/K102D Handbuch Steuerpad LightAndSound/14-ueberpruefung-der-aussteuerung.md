# Überprüfung der Aussteuerung

Öffnen Sie den LiveDaten-Assistenten. Drücken Sie die Standlicht-Taste, der Prozentwert sollte ca. 86 % ± 4 % anzeigen. Drücken Sie die Sattelkupplungstaste, der Wert sollte etwa -86 % ± 4 % betragen.

Liegt einer der Werte über 90 %, reduzieren Sie die Aussteuerung (Setup-Taste und S3 bzw. S2 oder S1) und starten Sie den Einlernprozess erneut.
