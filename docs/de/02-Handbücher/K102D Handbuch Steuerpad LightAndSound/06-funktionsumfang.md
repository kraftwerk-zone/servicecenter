# Funktionsumfang

Die Tasten können je nach Betätigungsdauer unterschiedliche Funktionen auslösen:

- **Kurze Betätigung:** Orange Symbole
- **Lange Betätigung:** Weiße Symbole
- **Zusatzebene (umschaltbar mit linker oberer Taste):** Graue Symbole, jeweils kurz/lang

## Obere Tastenreihe

| Kurze Betätigung | Lange Betätigung |
| --- | --- |
| Padebene | Motor Start/Stop |
| Blinker links | Servo 1 |
| Warnblinker | Partymodus |
| Blinker rechts | Servo 2 |
| Hochschalten | Lichthupe |
| Runterschalten | Hupe |

## Untere Tastenreihe

| Kurze Betätigung | Lange Betätigung |
| --- | --- |
| Standlicht | Stützen hoch |
| Abblendlicht | Stützen runter |
| Fernlicht | Mulde hoch |
| Nebelscheinwerfer/-schlussleuchte | Mulde runter |
| Rundumlicht | Rampe hoch |
| Sattelkupplung | Rampe runter |

**Padebene:** Durch Drücken der linken oberen Taste wird auf die Zusatzfunktionenebene umgeschaltet. Dort können bis zu 22 Zusatzfunktionen (jeweils elf kurz/lang) gesteuert werden. Die Funktionen sind entgegen dem Uhrzeigersinn von links unten beginnend angeordnet (graue Symbole).
