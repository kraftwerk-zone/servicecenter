# Überprüfung der Aussteuerung

Öffnen Sie den LiveDaten-Assistenten. Drücken Sie die Standlicht-Taste, der Prozentwert sollte bei ca. 86 % (+/- 4 %) liegen. Drücken Sie die Sattelkupplungstaste, sollte der Wert bei -86 % (+/- 4 %) liegen.

Liegt einer der Werte über 90 %, reduzieren Sie die Aussteuerung (Setup-Taste und S3 bzw. S2 oder S1) und starten Sie das Einlernen der Kanäle erneut.
