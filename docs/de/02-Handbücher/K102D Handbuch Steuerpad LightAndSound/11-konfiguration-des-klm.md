# Konfiguration des KLM

![](images/image\_013.png)

image15.png

Das Steuerpad kann an den Kanälen K1 oder K2 verwendet werden. Verbinden Sie das Modell per USB, schalten Sie Fernsteuerung und Modell ein und starten Sie das ControlPanel.

Öffnen Sie die Systemeinstellungen über das Zahnradsymbol oder den Menüpunkt „System“. Achten Sie darauf, dass der KLM in der Auswahlliste oben ausgewählt ist.

Ändern Sie die Einstellung „Belegung K1“ oder „Belegung K2“ auf „Pad Light And Sound“.

**Tipp:** Ist für K1 die Ebenensteuerung aktiviert, ist die Belegung für K2 ausgegraut, da dieser Kanal für die Ebenenumschaltung benötigt wird. Soll das Pad an K2 verwendet werden, wählen Sie für K1 eine andere Belegung, z. B. „Benutzerdefiniert“.

Nach dem Speichern ist der KLM für das Steuerpad konfiguriert.

**Tipp:** Wenn Sie unsicher sind, an welchem Kanal das Pad angeschlossen ist, hilft die LiveDaten-Ansicht im ControlPanel.
