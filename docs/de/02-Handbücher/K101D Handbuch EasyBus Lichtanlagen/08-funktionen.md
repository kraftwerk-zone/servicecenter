# Funktionen

## Lichtfunktionen

| Funktion | Beschreibung |
| --- | --- |
| Blinker links/rechts | Automatische Rückstellung, optional automatische Aktivierung bei Lenkeinschlag (siehe Abschnitt 9.3 „Lenkungstyp“). Blinkfrequenz in den Systemeinstellungen einstellbar. |
| Warnblinker | Automatisch bei Rückwärtsfahrt aktiv. |
| Standlicht | ![](images/image\_009.png)  image9.png |
| Standlicht & Abblendlicht | Optional mit Xenoneffekt für das Abblendlicht. |
| Standlicht & Fernlicht | Optional mit Xenoneffekt für das Fernlicht. |
| Standlicht, Abblend- & Fernlicht | Optional mit Xenoneffekt für Abblend- oder Fernlicht. |
| Abblendlicht | Optional mit Xenoneffekt. |
| Abblendlicht, Fernlicht | Optional mit Xenoneffekt für Abblend- oder Fernlicht. |
| Fernlicht | Optional mit Xenoneffekt. |
| Nebelscheinwerfer | ![](images/image\_010.png)  image16.png |
| Expert: Frontlicht | Alle Kombinationen möglich. |
| Tagfahrlicht | Immer an, kann gedimmt werden, wenn Blinker und/oder Standlicht eingeschaltet sind. |
| Abbiegelicht / Kurvenlicht | Aktiv bei Blinkerbetätigung oder bei Lenkeinschlag über Schwellwert (siehe Systemeinstellungen). Optional mit Nachleuchtzeit. |
| Bremslicht | Intelligente Steuerung mit einstellbarer Empfindlichkeit und Nachleuchtzeit. |
| Rückfahrlicht | Einfacher Vorwärts-/Rückwärtsregler oder mit getrennter Bremse/Rückwärtsfahrt. |
| Rücklicht, Nebelschlussleuchte | Alle Kombinationen möglich. |
| Amerikanischer Blinker | Standlicht und Blinker über eine LED. |
| Amerikanisches Rücklicht | Standlicht, Blinker und Bremslicht über eine LED. |
| Rundumlicht Ein/Aus | Schaltet den Ausgang auf 100%, wenn Rundumlicht aktiviert ist. |
| Einfachschalter | Schaltet den Ausgang bei Aktivierung der angegebenen Funktion. |
| Doppelschalter | Schaltet Ausgang auf Helligkeit 1 oder 2 je nach Funktion 1 oder 2. |
| Expert: Multiblitzer | Konfigurierbar mit Startverzögerung, Anzahl und Dauer der Pulse. |
| Expert: Lauflicht | Konfiguration von Lauflicht über mehrere Ausgänge. |
| Expert: KnightRider | Bekannter Lauflichteffekt mit einstellbarer Helligkeit und Geschwindigkeit pro Ausgang. |

## Servofunktionen

| Funktion | Beschreibung |
| --- | --- |
| Proportional | Steuerung eines Servos/Motorreglers mit einem Proportionalkanal. |
| Proportional geschaltet | Steuerung eines Servos/Motorreglers mit Proportionalkanal nur bei aktiver Schaltfunktion (z.B. Lenkung abschalten bei Liftachse hoch). |
| Zwei Positionen | Servo fährt von Position 1 zu Position 2 bei Aktivierung der Schaltfunktion. |
| Drei Positionen | Servo fährt von Position 2 zu Position 1 oder 3 je nach Schaltfunktion 1 oder 2. |
| Sattelplatte | Steuerung der Sattelplatte. |
| Schaltgetriebe | Steuerung eines Schaltgetriebes. |

## Motorfunktionen

| Funktion | Beschreibung |
| --- | --- |
| Proportional | Steuerung eines Motors mit einem Proportionalkanal. |
| Proportional geschaltet | Steuerung des Motors nur bei aktiver Schaltfunktion. |
| Sattelstützen | Motorsteuerung bei Auswahl der Sattelstützenfunktion. |
| Mulde | Motorsteuerung bei Muldenfunktion. |
| Kipper | Motorsteuerung bei Kippfunktion. |
