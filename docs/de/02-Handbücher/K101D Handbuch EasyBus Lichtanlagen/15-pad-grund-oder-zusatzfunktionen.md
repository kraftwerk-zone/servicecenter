# Pad Grund- oder Zusatzfunktionen

Die Pads codieren Tastendrücke in zwei Auf- und Abwärtsbewegungen. Der KLM empfängt diese Signale, dekodiert sie und steuert die entsprechenden Funktionen. Das Pad wird an einen Steuergeber im Fernsteuersender angeschlossen, der Kanal an K1 oder K2 des KLM.

Für das Einlernen der Kanäle muss das Pad in den Setup-Modus versetzt werden, indem beide Setup-Tasten gleichzeitig gedrückt werden. Das ControlPanel bietet einen Assistenten, der durch den Prozess führt.
