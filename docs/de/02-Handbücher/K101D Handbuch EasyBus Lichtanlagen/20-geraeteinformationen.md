# Geräteinformationen

Zeigt grundlegende Informationen zu Modulen. Mit „Zeige Gerät“ werden die ersten drei Ausgänge eingeschaltet, praktisch bei mehreren Universalerweiterungen ohne Namensvergabe.

Mit „Reset“ wird das Modul neu gestartet (vergleichbar mit Ab- und Anstecken). „Werkseinstellungen“ setzt auf Werkseinstellungen zurück, „Update“ erzwingt ein Update.

Bei verfügbarer neuer Version weist das ControlPanel beim Start darauf hin und bietet das Update an. Manuelles Update ist über Geräteinformationen möglich.
