# Einstellmöglichkeiten

Unsere EasyBus Systeme bieten zwei Arten von Einstellungen:

- **Systemeinstellungen:** Gilt für das gesamte Modul oder System (z.B. Blinkintervall, Empfängertyp, Bremslichttyp).
- **Ausgangsseinstellungen:** Gilt nur für den jeweiligen Ausgang (siehe Abschnitt 7 „Funktionen“).

## Systemeinstellungen öffnen

Öffnen Sie die Systemeinstellungen über das Zahnradsymbol oder im Menü unter „System“ > „Systemeinstellungen“.

| Name | Beschreibung |
| --- | --- |
| Startverzögerung [s] | Verzögert den Systemstart, um Anlaufzeit der Fernsteuerung zu berücksichtigen. |
| Einlernmodus [aktiv] | Aktiviert/deaktiviert den manuellen Einlernmodus (Gas- und Lenkknüppel in Endausschlag). Einlernen über ControlPanel bleibt aktiv. |
| Infrarot [ein] | Schaltet Ausgang 8 auf Infrarotbetrieb um. |
| Blinker Ein/Aus | Definiert Ein- und Auszeit des Blinkers in 10 ms Schritten. |
| Empfängertyp | Wahl der Ansteuerungsart (siehe Abschnitt 8). |
| Fahrtenreglertyp | Wahl des Fahrtenreglertyps (siehe Abschnitt 9.1). |
| Warnblinker beim Rückwärtsfahren | Automatische Aktivierung des Warnblinkers bei Rückwärtsfahrt. |
| Bremslicht Sensitivität | Empfindlichkeit des intelligenten Bremslichts. |
| Bremslicht Nachleuchten | Dauer des Nachleuchtens des Bremslichts. |
| Lenkungstyp | Art der Blinkerschaltung (manuell oder automatisch). |
| Blinker Schwellwert | Schwellenwert für automatisches Blinken bei Kurvenfahrt. |
| Kurvenlichttyp | Verhalten des Kurvenlichts. |
| Motor läuft sofort [aktiv] | Steuert, ob der Motor beim Einschalten sofort startet. |
| Motor Start Dauer | Dauer des Motorstart-Effekts (z.B. Flackern). |
| Pad Alles Ein/Aus | Konfiguration der „Alles Ein/Aus“-Taste bei Steuerpads. |
| Pad Light&Sound Fernlicht | Siehe Handbuch Steuerpad Licht & Sound. |
| Pad Light&Sound Nebelscheinwerfer | Siehe Handbuch Steuerpad Licht & Sound. |
| Belegung K1 – K12 | Benutzerdefinierte Belegung der Kanäle (siehe Abschnitt 13.6). |
