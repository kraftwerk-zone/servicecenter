# Geräteinformationen

Grundlegende Informationen zu einzelnen Modulen werden angezeigt. Mit „Zeige Gerät“ werden die ersten drei Ausgänge eingeschaltet, was bei mehreren Universalerweiterungen hilfreich ist.

Mit „Reset“ wird das Modul neu gestartet, mit „Werkseinstellungen“ auf Werkseinstellungen zurückgesetzt und mit „Update“ ein Update erzwungen.

Bei verfügbaren Updates wird beim Start des ControlPanels darauf hingewiesen. Manuelle Updates sind über die Geräteinformationen möglich.
