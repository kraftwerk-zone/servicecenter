# Updates

Nach dem Verbinden mit dem ControlPanel werden verfügbare Updates für Busmodule angeboten.

![](images/image\_052.png)

image6.png

**Update-Hinweise:**

- Führen Sie Updates immer nur für ein Modul gleichzeitig durch.
- Vermeiden Sie Stromunterbrechungen während des Updates.
- Schließen Sie ggf. Servos und Motoren ab, um Störungen zu vermeiden.
- Starten Sie Modell und ControlPanel regelmäßig neu, insbesondere bei ungewöhnlichen Verzögerungen.
