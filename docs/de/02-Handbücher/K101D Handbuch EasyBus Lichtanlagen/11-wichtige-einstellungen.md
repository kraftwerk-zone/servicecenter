# Wichtige Einstellungen

## Motor läuft sofort

Je nach Soundmodul muss die Systemeinstellung „Motor läuft sofort [aktiv]“ angepasst werden:

- Soundmodul startet Motor sofort: Haken setzen.
- Motor muss manuell gestartet werden: Haken entfernen.

**Vorsicht:** Der Fahrmotor ist in jedem Fall aktiv!

## Lichter flackern

Jede Beleuchtungsplatine und Schalterweiterung (KLB) besitzt Systemeinstellungen für „Starteffekt“ (Flackern beim Start) und „Motor Stop Lichtreduktion“ (Absenkung bei Motorstopp). Verwenden Sie letztere Option nur, wenn keine Komponenten mit integrierter Elektronik angeschlossen sind.

Die Dauer des Flackerns kann mit „Motor Start Dauer“ an die Startsound-Dauer angepasst werden.
