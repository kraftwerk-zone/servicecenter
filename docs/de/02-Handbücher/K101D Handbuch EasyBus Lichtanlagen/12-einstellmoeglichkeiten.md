# Einstellmöglichkeiten

Unsere EasyBus Systeme bieten zwei Einstellungsarten:

- **Systemeinstellungen:** Gilt für das gesamte Modul oder System (z.B. Blinkintervall, Empfängertyp, Bremslichttyp).
- **Ausgangsseinstellungen:** Gilt nur für den jeweiligen Ausgang (siehe Kapitel 7 „Funktionen“).

## Systemeinstellungen öffnen

Über das Zahnradsymbol oder das Menü „System“ → „Systemeinstellungen“.
