# Bestellung der Premiumsounds

Premiumsounds können bequem über unseren Online-Shop bestellt werden. Im Reiter **„Sounds“** finden Sie die aktuelle Auswahl an verfügbaren Premiumsounds.

![](images/image\_001.png)

image1.png

Wichtig: Jeder Sound kann nur auf einem einzelnen Soundmodul installiert werden. Falls Sie den gleichen Sound auf mehreren Modulen verwenden möchten, bieten wir attraktive Preisstaffeln an, um Ihnen Kosten zu sparen.

![](images/image\_002.png)

image2.png

Die Bestellung erfolgt wie gewohnt über den Warenkorb und wird nach Abschluss der Bezahlung automatisch bearbeitet. Es fallen keine Versandkosten an.
